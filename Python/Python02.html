一、Python 序列

1.1 概述

  • Python 序列（Sequence）是指按特定顺序依次排列的一组数据，它们可以占用一块连续的内存，也可以分散到多块内存中。序列类型包括 列表（<class 'list'>）、元组（<class 'tuple'>）、字典（<class 'dict'>）、集合（<class 'set'>）和 字符串（<class 'str'>）。

  • 列表和元组相似，它们都按顺序保存元素，所有的元素占用一块连续的内存，每个元素都有自己的索引，因此列表和元组的元素都可以通过索引（index）来访问。它们的 区别 在于：列表可以修改，而元组不可修改。

  • 字典和集合存储的数据都是无序的，每份元素占用不同的内存，其中字典元素以 key-value 的形式保存。

  • 集合和字典不支持索引、切片、相加和相乘操作。

  • 字符串也是一种常见的序列，它也可以直接通过索引访问字符串内的字符。

1.2 序列索引

  • 序列中，每个元素都有属于自己的编号（索引）：从起始元素开始，索引值从 0 开始递增。

  • Python 支持索引值是 负数，此类索引是从右向左计数：从最后一个元素开始，索引值从 -1 开始递减。

1.3 序列切片

  • 切片操作是访问序列中元素的另一种方法，它可以访问一定范围内的元素，通过切片操作，可以 生成一个新的序列。

  • 序列实现切片操作的语法格式：sname[start : end : step]

    • sname：表示序列的名称；

    • start：表示切片的开始索引位置（包含该位置），可以为负。可选，默认为 0，即序列开始。

    • end：表示切片的结束索引位置（不包含该位置），可以为负。可选，默认为序列长度。

    • step：表示在切片过程中，隔几个存储位置（包含当前位置）取一次元素，可选，如果 step 的值大于 1，则在进行切片操作时，会 “跳跃式” 的取元素。

      • step 为正表示从左向右，要求 start < end；step 为负表示从右向左，要求 start > end。

  • 当 start 或 end 超出有效索引范围​时，切片操作不会抛出异常，而是仅保留有效索引范围的值。

1.4 序列相加

  • Python 中，支持两种 类型相同 的序列使用 “+” 运算符做相加操作，它会将两个序列进行 连接，但不会去除重复的元素且 不改变序列类型。

1.5 序列相乘

  • Python 中，使用数字 n 乘以一个序列会生成新的序列，其内容为原来序列被重复 n 次的结果。

  • 特殊地，列表类型在进行乘法运算时，可以实现初始化指定长度列表的功能。例：list = [None] * 5 将创建一个长度为 5的列表，列表中的每个元素都是 None，即 [None, None, None, None, None]。

1.6 检查元素是否为序列成员

  • Python 中，可以使用 in 关键字检查某元素是否为序列成员，格式：value in sequence，其中 value 表示要检查的元素，sequence 表示指定的序列。

    • 集合和字典支持 in 操作符。

    • Python 字典的 in 操作符用于判断 键 是否存在于字典中，如果存在返回 true，否则返回 false。

  • not in 关键字和 in 关键字用法相同，但功能恰好相反的，它用来检查某个元素是否不包含在指定的序列中。

二、list 列表

2.1 概述

  • 类型 <class 'list'>。

  • Python 中没有数组，但是加入了更加强大的列表。

  • 从 表现形式 上看，列表会将所有元素都放在一对中括号 [ ] 里面，相邻元素之间用逗号 , 分隔：
    [element1, element2, ... , elementn]，其中 element1 ~ elementn 表示列表中的元素，个数没有限制，数据类型只要是 Python 支持的就可以。

  • 从 存储内容 上看，列表可以存储数字、字符串、列表、元组等任何类型的数据，并且同一个列表中元素的类型也可以不同。

    • 在使用列表时，虽然可以将不同类型的数据放入到同一个列表中，但通常情况下不这么做，同一列表中只放入同一类型的数据，这样可以提高程序的可读性。

2.2 创建列表

  • 创建列表的方法可分为两种：

    • 使用 [ ] 直接创建列表：listname = [element1, element2, ... , elementn]，其中 listname 表示变量名，element1 ~ elementn 表示列表元素。可以创建空列表，例：emptylist = []。

    • 使用内置函数 list() 创建列表，它可以将其它数据类型转换为列表类型。

2.3 访问列表元素

  • 列表是 Python 序列的一种，我们可以使用 索引（index）访问 列表中的某个元素（得到的是一个元素的值），也可以使用 切片访问 列表中的一组元素（得到的是一个新的子列表）。

    • 使用索引访问列表元素的格式为：listname[i]，其中 listname 表示列表名字，i 表示索引值。列表的索引可以是正数，也可以是负数。

    • 使用切片访问列表元素的格式为：listname[start : end : step]，其中 listname 表示列表名字，start 表示起始索引（包含该位置），end 表示结束索引（不包含该位置），step 表示步长。

2.4 删除列表

  • 对于已经创建的列表，如果不再使用，可以使用 del 关键字将其删除。格式为：del listname，其中 listname 表示要删除列表的名称。

2.5 列表方法

  • Python 列表的数据类型为 list，我们可使用 dir(list) 来查看该类型包含哪些方法。

2.5.1 列表添加元素

（1）append() 方法

  • append() 方法用于在 列表末尾 追加元素，格式：listname.append(obj)，其中 listname 表示要添加元素的列表，obj 表示要添加到列表末尾的数据，它可以是单个元素、列表、元组等。

  • 使用 append() 方法将 列表 或 元组 添加到列表末尾时，此方法会将它们作为一个 整体 添加到列表中，从而形成包含列表或元组的新列表。

（2）extend() 方法

  • extend() 方法用于在 列表末尾 追加元素，格式：listname.extend(obj)，其中 listname 表示要添加元素的列表，obj 表示要添加到列表末尾的数据，它可以是单个元素、列表、元组等，但 不能是单个数字。

  • extend() 和 append() 的不同之处在于：extend() 不会将列表或者元祖视为一个整体，而是将它们包含的元素 逐个添加 到列表中。

（3）insert() 方法

  • append() 和 extend() 方法只能在列表末尾插入元素，如果希望在 列表中间 某个位置插入元素，那么可以使用 insert() 方法。格式：listname.insert(index , obj)，其中 index 表示指定位置的索引值，obj 表示要添加到列表末尾的数据，它可以是单个元素、列表、元组等。insert() 方法会将 obj 插入到 listname 列表第 index 个元素的位置。

  • 使用 insert() 方法将列表或元祖插入列表时，insert() 方法也会将它们作为一个 整体 插入到列表中，这一点和 append() 方法是一样的。

2.5.2 列表删除元素

（1）del 关键字

  • del 是 Python 中用来执行删除操作的关键字，它不仅可以删除整个列表，还可以删除列表中的元素。

    • 使用 del 删除列表中的 单个元素，格式：del listname[index]，其中 listname 表示列表名称，index 表示元素的索引值。

    • 使用 del 删除列表中的 一段连续元素，格式：del listname[start : end]，其中 start 表示起始索引，end 表示结束索引。del 会删除从索引 start 到 end 之间的元素，不包括 end 索引的元素。

（2）pop() 方法

  • pop() 方法用于删除列表中指定索引处的元素，格式：listname.pop(index)，其中 listname 表示列表名称，index 表示索引值。

    • 如果不写 index 参数，pop() 方法默认删除列表中的最后一个元素，类似于数据结构中的 “出栈” 操作。
（3）remove() 方法
  • remove() 方法用于移除列表中第一个与指定值相匹配的元素，格式：listname.remove(obj)，其中 obj 表示要移除的元素。注：要移除的元素不存在会报错 ValueError: list.remove(x): x not in list 。
（4）clear() 方法
  • clear() 方法用于删除列表中的所有元素，即清空列表，类似于 del a[:]，格式：listname.clear()。

2.5.3 列表修改元素

（1）修改单个元素
  • 使用索引得到列表元素，直接通过 = 重新赋值来改变元素的值。
（2）修改一组元素

  • Python 支持通过切片语法为列表修改元素，即使用切片重新赋值来改变一组元素的值。如果不指定步长（step 参数），Python 就不要求新赋值的元素个数与切片获取到的元素个数相同。

    • 如果为切片赋值空列表，就相当于删除一组元素。

    • 如果对空切片赋值，就相当于插入一组新的元素。

2.5.4 列表查找元素

（1）count() 方法

  • count() 方法用于统计某个元素在列表中出现的次数，格式：listname.count(obj)，其中 listname 代表列表名，obj 表示要统计的元素。

  • 如果 count() 返回 0，就表示列表中不存在该元素，所以 count() 也可以用来判断列表中的某个元素是否存在。

（2）index() 方法

  • index() 方法用于查找某个元素在列表中出现的位置（索引），格式：listname.index(obj, start, end)，其中 listname 表示列表名称，obj 表示要查找的元素，start 表示起始位置（包含），end 表示结束位置（不包含）。

  • 如果要查找的元素不存在，则会报错：ValueError: x is not in list，建议查找前使用 count() 方法进行判断。

三、tuple 元组

3.1 概述

  • 类型 <class 'tuple'>。

  • 元组（tuple）是 Python 中另一个重要的序列结构，和列表类似，元组也是由一系列按特定顺序排序的元素组成。

  • 元组和列表（list）的 区别：列表的元素是可以更改的，包括修改、删除和插入元素，所以 列表是可变序列。然而，元组一旦被创建，它的元素就不可更改了，所以 元组是不可变序列。

  • 从 表现形式 上看，元组的所有元素都放在一对小括号 ( ) 中，相邻元素之间用逗号 , 分隔，格式：(element1, element2, ... , elementn)，其中 element1 ~ elementn 表示列表中的元素，个数没有限制，数据类型只要是 Python 支持的就可以。

  • 从 存储内容 上看，元组可以存储数字、字符串、列表、元组等任何类型的数据，并且在同一个元组中，元素的类型可以不同。

3.2 创建元组

3.2.1 使用 ( ) 创建元组

  • 通过 ( ) 创建元组后，一般使用 = 将它赋值给某个变量，格式：tuplename = (element1, element2, ..., elementn)，其中 tuplename 表示元组名，element1 ~ elementn 表示元组的元素。

  • 在 Python 中，元组通常都是使用一对小括号将所有元素包围起来的，但 小括号不是必须的，只要将各元素用逗号隔开，Python 就会将其视为元组。

    • 当创建的元组中只有一个字符串类型的元素时，该元素后面必须要加一个逗号,，否则 Python 解释器会将它视为字符串。

3.2.2 使用 tuple() 方法创建元组

  • Python 提供了一个内置函数 tuple() 将其它数据类型转换为元组类型，格式：tuple(data)，其中 data 表示可以转化为元组的数据，包括字符串、元组、range 对象等。

3.3 访问元组

  • 和列表一样，可以使用 索引（index）访问 元组中的某个元素（得到的是一个元素的值），也可以使用 切片访问 元组中的一组元素（得到的是一个新的子元组）。

    • 使用 索引访问 元组元素的格式：tuplename[i]，其中 tuplename 表示元组名字，i 表示索引值。元组的索引可以是正数，也可以是负数。

    • 使用 切片访问 元组元素的格式：tuplename[start : end : step]，其中 start 表示起始索引（包含该位置），end 表示结束索引（不包含该位置），step 表示步长。

3.4 删除元组

  • 对于已经创建的元组，如果不再使用，可以使用 del 关键字将其删除。格式为：del tuplename，其中 tuplename 表示要删除元组的名称。注：元组不支持单个元素删除。

3.5 元组查找元素

3.5.1 count() 方法

  • count() 方法用于统计某个元素在元组中出现的次数，格式：tuplename.count(obj)，其中 tuplename 代表元组名，obj 表示要统计的元素。

  • 如果 count() 返回 0，就表示元组中不存在该元素，所以 count() 也可以用来判断元组中的某个元素是否存在。

3.5.2 index() 方法

  • index() 方法用于查找某个元素在元组中出现的位置（索引），格式：listname.index(obj[, start, end])，其中 listname 表示元组名称，obj 表示要查找的元素，start 表示起始位置（包含），end 表示结束位置（不包含）。

  • 如果要查找的元素不存在，则会报错：ValueError: tuple.index(x): x not in tuple，建议查找前使用 count() 方法进行判断。

四、dict 字典

4.1 概述

  • 类型 <class 'dict'>。

  • Python 字典（dict）是一种 无序的、可变的 序列，它的元素以 “键值对（key-value）” 的形式存储。相对地，列表（list）和元组（tufromkeysple）都是有序的序列，它们的元素在底层是挨着存放的。

  • 字典类型是 Python 中唯一的 映射类型，即通过一个元素，可以唯一找到另一个元素。

  • 字典中，习惯将各元素对应的 索引 称为 键（key），各个键对应的 元素 称为值（value），键及其关联的值称为 “键值对”。

  • 字典中，键必须唯一，但值可不唯一，即为不同键赋同一值。值可以取任何数据类型，但键必须是不可变类型，例：字符串、数字或元组等。

4.2 创建字典

4.2.1 使用 { } 创建字典

  • 由于字典中每个元素都包含 两部分，分别是 键（key）和 值（value），因此在创建字典时，键和值之间使用 冒号 : 分隔，相邻元素之间使用 逗号 , 分隔，所有元素放在大括号 { } 中。格式：dictname = {'key1':'value1', 'key2':'value2', ..., 'keyn':valuen}，其中 dictname 表示字典变量名，'keyn' : valuen 表示各个元素的键值对。

4.2.2 使用 fromkeys() 方法创建字典

  • 使用 dict 字典类型提供的 fromkeys() 方法创建带有默认值的字典，格式：dictname = dict.fromkeys(list，value=None)，其中 list 参数表示字典中所有键的列表（list），value 参数表示默认值，如果不写，则为空值 None。这种创建方式通常用于 初始化字典，设置 value 的默认值。

4.2.3 使用 dict() 内置函数创建字典

（1）方式一：使用关键字参数创建

  • 格式：dict(**kwarg)，双星号（**）参数 **kwargs 要求将参数以键值的形式导入。

  • 示例：dict(name='dong', age=18, height=226)，使用此方式创建字典时，字符串键不能带引号。

（2）方式二：使用映射对象创建字典

  • 格式：dict(mapping, **kwarg)，与 dict(**kwarg) 函数不一样的地方是第一个参数为 mapping 类型的数据。mapping 是一种关联式的容器类型，它存储了对象与对象之间的映射关系。通过 zip 函数、map 函数等可以直接生成 mapping 类型的数据。

  • 示例：

（3）方式三：使用可迭代对象创建字典

  • 格式：dict(iterable, **kwarg)，与 dict(**kwarg) 函数不一样的地方是第一个参数为 iterable 类型的数据。iterable 数据中每一项的长度必须为 2，其中第 1 个元素作为键，第 2 个元素作为值。否则抛出异常：ValueError: dictionary update sequence element #0 has length 1; 2 is required。

  • 列表和元组是通过下标来访问元素的，而字典是 通过键来访问对应的值。格式：dictname[key]，其中 dictname 表示字典的名称，key 表示键名。注：如果键不存在，会抛出 KeyError 异常。示例：

  • 字典提供了 get() 方法来获取指定键对应的值（推荐）。当指定键不存在时，get() 方法不会抛出异常。格式：dictname.get(key[,default])，其中 dictname 表示字典的名称；key 表示指定键；default 表示指定键不存在时，此方法返回的默认值，如果不指定，会返回 None。示例：

4.4 删除字典

  • 对于已经创建的字典，如果不再使用，可以使用 del 关键字将其删除。格式为：del dictname，其中 dictname 表示要删除字典的名称。

4.5 键值对操作

4.5.1 添加键值对

  • 添加新的键值对直接给不存在的 key 赋值即可，语法格式：dictname[key] = value，其中 dictname 表示字典的名称；key 和 value 表示新添加的键值对。

4.5.2 修改键值对

  • 字典中键（key）的名字不能被修改，只能修改值（value）。

  • 字典中各元素的键必须是唯一的，因此，如果新添加元素的键与已存在元素的键相同，那么键所对应的值就会被新的值替换掉，以此达到修改元素值的目的。

4.5.3 删除键值对

  • 如果要删除字典中的键值对，还是可以使用 del 语句，格式：del dictname[key]，其中 dictname 表示字典的名称，key 表示键。

4.5.4 判断字典中是否存在指定键值对

  • 如果要判断字典中是否存在指定键值对，首先应判断字典中是否有对应的键。

  • 判断字典是否包含指定键值对的键，可以使用 in 或 not in 运算符。示例：

  • 通过 in（或 not in）运算符，我们可以判断字典中是否包含某个键，如果存在，可以通过键获取对应的值，判断是否有指定的键值对存在。
