一、Python 解释器

  • Python 是一种高级计算机语言，设计它们的目的是为了方便程序员理解和使用。但严格意义上来说，计算机只能理解一种非常低级的计算机语言，也就是 机器语言。

  • 我们需要设计一种方法，将高级语言翻译成计算机可以执行的机器语言。目前，有两种方法可以实现，分别是 编译器 和 解释器。

  • 使用 编译器 将自身转换成机器语言的高级语言，通常称为 编译型语言；而使 解释器 将自身转换成机器语言的高级语言，称为 解释型语言，Python 就是一种解释型编程语言。

  • 解释型语言 由 解释器 逐行对源码进行解释，一边解释一边执行，不会生成可执行程序。

  • Python 官方针对不同平台（例：Windows、MacOS、Linux）开发了不同的解释器，这些解释器必须要遵守同样的语法，识别同样的函数，完成同样的功能。只有这样，同一份代码才能在不同平台上拥有相同的执行结果。

  • Python 解析器种类

    • CPython，C 语言开发的解释器 [官方]，应用广泛的解释器。

    • IPython，基于 CPython 的一种 交互式 解释器。

    • PyPy，基于 Python 语言开发的解释器，解析效率低。

    • JPython，运行在 Java 平台的解释器，直接把 Python 代码编译成 Java 字节码执行。

    • IronPython，运行在微软 .Net 平台的解释器，直接把 Python 代码编译成 .Net 字节码执行。

  • Python 解析器下载

    • 选择 Windows x86-64 executable installer 版本。

  • Python 解析器安装

三、Python 变量

2.1 强、弱类型语言

  • Python、JavaScript、PHP 等脚本语言都是 弱类型语言，这种语言有下面两种特点：

    • 变量无须声明就可以直接赋值，对一个不存在的变量赋值就相当于定义了一个新变量。

    • 变量的数据类型可以随时改变，例：同一变量可以先被被赋值为整数，然后再被赋值为字符串。

  • 弱类型不代表没有数据类型，弱类型是说在声明变量时无须指明数据类型，但是在编程语言的内部仍然是有数据类型的。

  • 我们可以使用 type() 内置函数来检测某个变量或者表达式的类型。

2.2 Python 变量的定义

  • 变量是存储数据的一块内存地址的别名。

  • 每个变量都拥有独一无二的名字，通过变量的名字就能找到变量中的数据。

  • 变量存储的数据是临时的。

2.3 Python 变量的赋值

  • 将数据放入变量的过程就叫做赋值。Python 使用等号 = 作为赋值运算符，格式：name = value，其中 name 表示变量名，value 表示变量值，也就是要存储的数据。

  • 变量的值可以随时被修改，只要重新赋值即可，可以将不同类型的数据赋值给同一个变量。

  • 变量名要遵守 Python 标识符命名规范，还要避免和 Python 内置函数及 Python 保留字重名。

  • 标识符 主要作用是作为变量、函数、类、模块及其他对象的名称，Python 标识符命名规范：

    • 标识符是由字母（A ~ Z 和 a ~ z）、数字和下划线（_）组成。

    • 标识符第一个字符不能是数字。

    • 标识符不能和 Python 中的保留字相同。

    • 标识符严格区分大小写。

    • 以下划线开头的标识符通常有特殊含义，例：

      • 以 单下划线开头 的标识符（例：_width）表示不能直接访问的 类属性，其无法通过 from ... import * 的方式导入。

      • 以 双下划线开头 的标识符（例：__add）表示 类的私有成员。

      • 以 双下划线开头和结尾 的标识符（例：__init__），是 专用标识符。

  • 保留字 是 Python 语言中一些已经被赋予特定意义的单词，所以不能使用这些保留字作为标识符给变量、函数、类、模块及其他对象命名。Python 中一共有 33 个保留字，如下表所示：

  • 内置函数 是 Python 解释器自带的函数，这些函数可以直接使用，不需要导入某个模块。下表列出了 Python 3.x 中的所有内置函数。

  • Python 变量的命名习惯：

    • Python 变量命名一定要做到见名知义。

    • 当标识符用作 模块名 或 包名 时，应全部使用小写字母，尽量不使用下划线。

    • 当标识符用作 类名 时，应采用单词首字母大写的形式（大驼峰）。

    • 当标识符用作 函数名 或 变量名 时，应全部使用小写字母，多个单词之间使用下划线分割。

    • 当标识符用作 常量 时，应全部使用大写字母，多个单词之间使用下划线分割。

四、Python 基本数据类型

  • Python 基本数据类型一般分为 6 种：数值（Numbers）、字符串（String）、列表（List）、元组（Tuple）、字典（Dictionary）、集合（Set）。

  • Python 的六个标准数据类型中：

    • 不可变数据（3 个）：Number（数字）、String（字符串）、Tuple（元组）。

    • 可变数据（3 个）：List（列表）、Dictionary（字典）、Set（集合）。

  • Python 内置的 type() 函数可以用来查询变量所指向的对象类型。

4.1 Numbers 数值

  • Numbers 数字类型是不允许改变的，这就意味着如果改变数字数据类型的值，将重新分配内存空间。

  • Python 中数字类型有 3 种：整型数（int）、浮点型数（float）、复数（complex）。其中，整型数又可以分为整型（int）和布尔型（bool）。

4.1.1 int 整型数

（1）int 整型

  • 类型 <class 'int'>。

  • Python 整数不分类型，或者说它只有一种类型的整数。Python 整数的取值范围是无限的，不管多大或者多小的数字。

  • 当所用数值超过计算机自身的计算能力时，Python 会自动转用高精度计算（大数计算）。

  • 在 Python 中，可以使用多种进制来表示整数：

    • 十进制形式：常见的整数形式，由 0 ~ 9 共十个数字排列组合而成。

    • 二进制形式：由 0 和 1 两个数字组成，书写时以 0b 或 0B 开头。

    • 八进制形式：由 0 ~ 7 共八个数字组成，书写时以 0o 或 0O 开头。

    • 十六进制形式：由 0 ~ 9 十个数字和 A ~ F（或 a ~ f）六个字母组成，书写时以 0x 或 0X 开头。

  • 为了提高数字的的可读性，Python 3.x 允许使用下划线 _ 作为数字（包括整数和小数）的分隔符。通常每隔三个数字添加一个下划线，类似于英文数字中的逗号。例：num = 1_234.123_456。

（2）bool 布尔型

  • 类型 <class 'bool'>。

  • Python 提供了 bool 类型来表示真（对）或假（错），其中 真（对）使用 True 来代表、假（错）使用 False 来代表。

  • True 和 False 是 Python 中的关键字，要注意字母的大小写。

  • 在 Python 中，bool 是 int 的子类，bool.__base__ 返回的 基类 是 <class 'int'>，True 和 False 可以和数字直接相加，类似于数值 1 和 0。在实际使用中，True == 1、False == 0 会返回 True。

4.1.2 float 浮点型数

  • 类型 <class 'float'>。

  • Python 只有一种小数类型，就是 float。书写时必须包含一个小数点，否则会被当作整数处理。

  • Python 中的小数有两种书写形式：

    • 十进制形式：常见的浮点数形式。

    • 指数形式：写法为 aEn 或 aen，整个表达式等价于 a×10na×10^na×10n。例：2.1E5=2.1×1052.1E5 = 2.1×10^52.1E5=2.1×105。

4.1.3 complex 复数

  • 类型 <class 'complex'>。

  • 复数（complex）是 Python 的内置类型，直接书写即可。

  • 复数由实部（real）和虚部（imag）构成，在 Python 中，复数的虚部以 j 或者 J 作为后缀，具体格式为：a + bj，其中 a 表示实部，b 表示虚部。例：12 + 0.2j。

  • 使用 complex 函数可以表示复数，例：complex(1, 2) 表示复数 1 + 2j。

4.1.4 数字类型转换

  • int(x)：内置函数，将 x 转换为一个整数。

  • float(x)：内置函数，将 x 转换为一个浮点数。

  • bool(x)：内置函数，将 x 转换 bool 类型。

  • complex(x)：内置函数，将 x 转换为一个复数，实数部分为 x，虚数部分为 0。

  • complex(x, y)：内置函数，将 x 和 y 转换到一个复数，实数部分为 x，虚数部分为 y。

  • chr(x)：内置函数，将一个整数转换为一个字符。

  • ord(x)：内置函数，将一个字符转换为它对应的 ASCII 数值或 Unicode 数值。

  • hex(x)：内置函数，将一个整数转换为一个 16 进制的字符串。

  • oct(x)：内置函数，将一个整数转换为一个 8 进制的字符串。

4.2 String 字符串

4.2.1 普通字符串

  • 若干个字符的集合就是一个字符串（String），Python 中的字符串使用双引号 " " 或单引号 ' ' 创建，双引号和单引号没有区别。

  • 当字符串内容中出现 引号 时，需要对其进行特殊处理，有两种处理方法：

    • 在引号前面添加反斜杠 \ 对引号进行转义，将引号作为普通文本。

    • 使用不同的引号创建字符串，如果字符串内容中出现了单引号，可以使用双引号创建字符串，反之亦然。

  • 要想 换行书写 一个比较长的字符串，可以在行尾添加反斜杠 \ 对字符串内容实现换行。

4.2.2 长字符串

  • Python 提供了 长字符串 的使用。所谓长字符串，就是可以直接换行（不用加反斜杠 \）书写的字符串。

    • Python 长字符串由三个双引号 """ 或者三个单引号 ''' 创建。

    • 长字符串中的换行、空格、缩进等空白符都会原样输出。

    • 在长字符串中放置单引号或者双引号不会导致解析错误。

    • 如果长字符串没有赋值给任何变量，那么这个长字符串就相当于多行注释。

4.2.3 原始字符串

  • 原始字符串 就是在普通字符串或者长字符串的开头加上 r 前缀。

    • 示例：str = r"F:\anaconda\envs\py39torch\Lib\site-packages\__pycache__"

    • 如果字符串中出现引号，程序需要对引号进行转义（加转义字符 \），但是和普通字符串不同的是，此时用于转义的反斜杠会变成原始字符串内容的一部分。即 print(r'I\'m a great coder!') 输出结果为 I\'m a great coder!

    • 原始字符串中的反斜杠仍然会对引号进行转义，因此原始字符串的结尾处不能是反斜杠，否则字符串结尾处的引号会被转义，导致字符串不能正确结束。

      • 在 Python 中有两种方式解决这个问题：

        • 改用长字符串的写法，不要使用原始字符串。

        • 单独书写反斜杠。例：print(r'F:\anaconda\envs' '\\')，输出结果 F:\anaconda\envs\，这是因为 Python 会自动将 print 中的两个字符串拼接在一起。

4.2.4 字符串转换

  • str(x)：内置函数，将对象转化为适于人类阅读的字符串。

  • repr(x)：内置函数，将对象转化为供解释器读取的字符串。可以将读取到的格式字符（例：换行符、制表符）转化为相应的转义字符。

  • eval(str)：用来执行一个字符串表达式，并返回表达式的值。

4.2.6 格式化输出

（1）格式化字符串

  • 格式化字符串 相当于一个字符串模板，可以放置一些转换说明符。转换说明符
    （Conversion Specifier）就是以 % 开头的占位符，用来对各种类型的数据进行格式化输出，它会被后面表达式（变量、常量、数字、字符串、加减乘除等各种形式）的值代替。

  • 转换说明符的语法格式：%[(name)][flags][width][.precision] typecode。

    • name：可选。表示参数的名称，后面的替换表达式需要使用字典格式，使用 name 时需要用括号包围。例：

    • flags：可选。表示对齐方式，指定宽度时默认右对齐，可以选择 +、-、' ' 或 0（允许多选）。其中，对于字符串，只能使用 - 标志，因为符号对于字符串没有意义，而补 0 会改变字符串的值。

      • + 表示输出的数字总要带着符号，正数带 +，负数带 -。

      • - 表示左对齐。

      • ' ' 为一个空格，表示在正数的左侧填充一个空格，从而与负数对齐。

      • 0 表示宽度不足时，使用 0 填充。

        • 对于数字，指定左对齐时，在右边补 0 是没有效果的，因为这样会改变数字的值。

    • width：可选。表示最小输出宽度，如果输出大于 width，则原数据宽度输出。

    • .precision：可选。表示小数点后精度。精度需要放在最小宽度之后，中间用点号 . 隔开：%width.precisionF；也可以不写最小宽度，只写精度：%.precisionF。

    • typecode：必选。表示输出数据的具体类型。

（2）format() 函数

  • 自 Python 2.6 版本开始，字符串类型（str）提供了 format() 方法对字符串进行格式化，语法：<模板字符串>.format(<逗号分隔的参数>)。

  • 模板字符串格式：XXX {[index][:[fill][align][sign][#][0][width][grouping_option][.precision] [type]]} XXX，其中 XXX 表示正常文本，会原样输出，{} 内部的字段会被 format() 中的参数替换。

    • index：可选。表示 : 后边设置的格式要作用到 format() 中第几个参数，参数的索引值从 0 开始。

      • 如果省略此选项，则会根据 format() 中参数的先后顺序自动分配。

      • 支持通过名称填充，format() 中的参数名称与 index 名称要对应。例：

      • 支持通过列表填充

      • 支持通过字典填充

    • fill：可选。表示空白处填充的字符。注：当填充字符为逗号（,）且作用于整数或浮点数时，该整数（或浮点数）会以逗号分隔的形式输出，例：1000000 会输出 1,000,000。

    • align：可选。表示数据的对齐方式。

      • <：数据左对齐。

      • >：数据右对齐。

      • =：数据右对齐，同时将符号放置在填充内容的最左侧，该选项只对数字类型有效。

      • ^：数据居中，此选项需和 width 参数一起使用。

    • sign：指定数字类型符号。

      • +：正数前加正号，负数前加负号。

      • -：正数前不加正号，负数前加负号。

      • ``（空格）：正数前加空格，负数前加负号。

    • #：对于二进制数、八进制数和十六进制数，使用此参数，各进制数前会分别显示 0b、0o、0x 或 0X 前缀；反之则不显示前缀。

    • 0：用 0 补足 width 宽度。如果设置了 fill，则使用 fill 填充。

    • width：可选。表示最小输出宽度，如果输出大于 width，则原数据宽度输出。

    • grouping_option：可选。表示千分位符，取值为 _ 或 ,。

    • .precision：可选。表示保留的小数位数，小数点不能省略。

    • type：指定输出数据的具体类型。

五、Python 输入/输出

5.1 input() 函数：获取用户输入的字符串

  • input() 是 Python 的内置函数，用于从控制台读取用户输入的内容。input() 函数总是以字符串的形式来处理用户输入的内容，所以用户输入的内容可以包含任何字符。

  • input() 函数的用法为：str = input(tipmsg)

    • str 表示一个字符串类型的变量，input 会将读取到的字符串放入 str 中。

    • tipmsg 表示提示信息，它会显示在控制台，如果不写 tipmsg，就不会有任何提示信息。

5.2 print() 函数：输出

  • 语法格式：print(value,...,sep='',end='\n',file=sys.stdout,flush=False)。

    • print 函数可以接受多个 value 参数，参数之间以逗号 , 分隔，因此 print 函数可以输出多个值。

    • 使用 print 函数输出多个变量时，默认以空格分隔多个变量，可通过 sep 参数修改分隔符。

    • 在默认情况下，print 函数输出之后会自动换行，这是因为 print 函数的 end 参数的默认值是 “\n”，这个 “\n” 就代表了换行。如果希望 print 函数输出之后不会换行，则重设 end 参数即可。例：

六、运算符

6.1 算术运算符

  • +、-、*、/、//（整除）、%（取模 / 取余）、**（幂运算/次方运算）

    • 当 + 用于数字时表示加法，但是当 + 用于字符串时，它还有拼接字符串的作用。

    • * 除了可以用作乘法运算，还可以用来重复字符串，即将 n 个同样的字符串连接起来。

    • // 表示整除，只保留结果的整数部分，直接舍弃小数部分，而不是四舍五入。

      • 如果有小数参与运算时，// 的结果为小数（小数部分为 0）；否则为整数。

    • / 、// 和 % 的除数为 0 将抛出 ZeroDivisionError 异常。

    • 只有当除数为负时，取模的结果才为负。换句话说，取模结果的正负和被除数没有关系，只由除数决定。

    • % 两边的数字都是整数时，取模的结果也是整数；只要有一个数字是小数，取模的结果就是小数。

    • 由于开方是次方的逆运算，所以可以使用 ** 运算符间接实现开方运算。例：81**(1/4) 结果为 3.0。

6.2 位运算符

  • &、|、^、~、<<、>>

6.3 比较运算符（关系运算符）

  • >、<、==、>=、<=、!=、is、is not

    • == 用来比较两个变量的值是否相等，而 is 则用来比对两个变量引用的是否是同一个对象。

6.4 逻辑运算符

  • and、or、not

    • 对于 and 运算符，两边的值都为真时最终结果才为真，只要其中有一个值为假，那么最终结果就是假，所以 Python 按照下面的规则执行 and 运算：

      • 如果左边表达式的值为假，不计算右边表达式，此时 and 会把左边表达式的值作为最终结果。

      • 如果左边表达式的值为真，and 会继续计算右边表达式的值，并将右边表达式的值作为最终结果。

    • 对于 or 运算符，两边的值都为假时最终结果才为假，只要其中有一个值为真，那么最终结果就是真，所以 Python 按照下面的规则执行 or 运算：

      • 如果左边表达式的值为真，不计算右边表达式，此时 or 会把左边表达式的值作为最终结果。

      • 如果左边表达式的值为假，or 会继续计算右边表达式的值，并将右边表达式的值作为最终结果。

6.5 三目运算符（三元运算符）

  • Python 没有引入 ? : 这个运算符，而是使用已有的 if else 关键字来实现相同的功能。

  • 格式：exp1 if contion else exp2

    • condition 是判断条件，exp1 和 exp2 是两个表达式。

    • 如果 condition 成立（结果为真），就执行 exp1，并把 exp1 的结果作为整个表达式的结果；如果 condition 不成立（结果为假），就执行 exp2，并把 exp2 的结果作为整个表达式的结果。
