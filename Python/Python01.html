一、Python 解释器

  • Python 是一种高级计算机语言，设计它们的目的是为了方便程序员理解和使用。但严格意义上来说，计算机只能理解一种非常低级的计算机语言，也就是 机器语言。

  • 我们需要设计一种方法，将高级语言翻译成计算机可以执行的机器语言。目前，有两种方法可以实现，分别是 编译器 和 解释器。

  • 使用 编译器 将自身转换成机器语言的高级语言，通常称为 编译型语言；而使 解释器 将自身转换成机器语言的高级语言，称为 解释型语言，Python 就是一种解释型编程语言。

  • 解释型语言 由 解释器 逐行对源码进行解释，一边解释一边执行，不会生成可执行程序。

  • Python 官方针对不同平台（例：Windows、MacOS、Linux）开发了不同的解释器，这些解释器必须要遵守同样的语法，识别同样的函数，完成同样的功能。只有这样，同一份代码才能在不同平台上拥有相同的执行结果。

  • Python 解析器种类

    • CPython，C 语言开发的解释器 [官方]，应用广泛的解释器。

    • IPython，基于 CPython 的一种 交互式 解释器。

    • PyPy，基于 Python 语言开发的解释器，解析效率低。

    • JPython，运行在 Java 平台的解释器，直接把 Python 代码编译成 Java 字节码执行。

    • IronPython，运行在微软 .Net 平台的解释器，直接把 Python 代码编译成 .Net 字节码执行。

  • Python 解析器下载

    • 官网：https://www.python.org/downloads/release/python-379/

    • 选择 Windows x86-64 executable installer 版本。

  • Python 解析器安装

    • 双击 python-3.7.9-amd64.exe 安装包。

    • 勾选 Add Python 3.7 to PATH，然后点击 Customize installation。

    • Python 最重要的两部分：Python 解释器 和 pip 包依赖管理工具。

    • 勾选 Install for all users →\rightarrow→ 自定义安装路径 →\rightarrow→ Install →\rightarrow→ Close。

    • 测试安装成功：命令提示符中输入 python，出现交互式语言即为安装成功，输入 exit() 退出。

三、Python 变量

2.1 强、弱类型语言

  • Python、JavaScript、PHP 等脚本语言都是 弱类型语言，这种语言有下面两种特点：

    • 变量无须声明就可以直接赋值，对一个不存在的变量赋值就相当于定义了一个新变量。

    • 变量的数据类型可以随时改变，例：同一变量可以先被被赋值为整数，然后再被赋值为字符串。

  • 弱类型不代表没有数据类型，弱类型是说在声明变量时无须指明数据类型，但是在编程语言的内部仍然是有数据类型的。

  • 我们可以使用 type() 内置函数类检测某个变量或者表达式的类型，

2.2 Python 变量的定义

  • 变量是存储数据的一块内存地址的别名。

  • 每个变量都拥有独一无二的名字，通过变量的名字就能找到变量中的数据。

  • 变量存储的数据是临时的。

2.3 Python 变量的赋值

  • 将数据放入变量的过程就叫做赋值。Python 使用等号 = 作为赋值运算符，格式：name = value，其中 name 表示变量名，value 表示变量值，也就是要存储的数据。

  • 变量的值可以随时被修改，只要重新赋值即可，可以将不同类型的数据赋值给同一个变量。

  • 变量名要遵守 Python 标识符命名规范，还要避免和 Python 内置函数及 Python 保留字重名。

  • 标识符 主要作用是作为变量、函数、类、模块及其他对象的名称，Python 标识符命名规范：

    • 标识符是由字母（A ~ Z 和 a ~ z）、数字和下划线（_）组成。

    • 标识符第一个字符不能是数字。

    • 标识符不能和 Python 中的保留字相同。

    • 标识符严格区分大小写。

    • 以下划线开头的标识符通常有特殊含义，例：

      • 以 单下划线开头 的标识符（例：_width）表示不能直接访问的 类属性，其无法通过 from ... import * 的方式导入。

      • 以 双下划线开头 的标识符（例：__add）表示 类的私有成员。

      • 以 双下划线开头和结尾 的标识符（例：__init__），是 专用标识符。

  • 保留字 是 Python 语言中一些已经被赋予特定意义的单词，所以不能使用这些保留字作为标识符给变量、函数、类、模块及其他对象命名。Python 中一共有 33 个保留字，如下表所示：

    • 注：Python 严格区分大小写，保留字也不例外。所以，False 是保留字，但 false 不是保留字。

  • 内置函数 是 Python 解释器自带的函数，这些函数可以直接使用，不需要导入某个模块。下表列出了 Python 3.x 中的所有内置函数。

    • 不要使用内置函数作为标识符，虽然这样做 Python 解释器不会报错，但这会导致同名的内置函数被覆盖，从而无法使用。

  • Python 变量的命名习惯：

    • Python 变量命名一定要做到见名知义。

    • 当标识符用作 模块名 或 包名 时，应全部使用小写字母，尽量不使用下划线。

    • 当标识符用作 类名 时，应采用单词首字母大写的形式（大驼峰），私有类使用一个下划线开头。

    • 当标识符用作 函数名 时，应全部使用小写字母，多个单词之间使用下划线分割。

      • 类的 私有函数 使用下划线（_）开头（该函数可被继承访问）。

      • 类的 私有函数 使用单下划线（_）开头（该函数不可被继承访问）。

    • 当标识符用作 变量名 时，应全部使用小写字母，多个单词之间使用下划线分割。

      • 类的 内部变量 使用单下划线（_）开头（该变量可被继承访问）。

      • 类的 私有变量 使用双下划线（__）开头（该变量不可被继承访问）

    • 当标识符用作 常量 时，应全部使用大写字母，多个单词之间使用下划线分割。

四、Python 基本数据类型

  • Python 基本数据类型一般分为 6 种：数值（Numbers）、字符串（String）、列表（List）、元组（Tuple）、字典（Dictionary）、集合（Set）。

  • Python 的六个标准数据类型中：

    • 不可变数据（3 个）：Number（数字）、String（字符串）、Tuple（元组）。

    • 可变数据（3 个）：List（列表）、Dictionary（字典）、Set（集合）。

  • Python 内置的 type() 函数可以用来查询变量所指向的对象类型。

4.1 Numbers 数值

  • Numbers 数字类型是不允许改变的，这就意味着如果改变数字数据类型的值，将重新分配内存空间。

  • Python 中数字类型有 3 种：整型数（int）、浮点型数（float）、复数（complex）。其中，整型数又可以分为整型（int）和布尔型（bool）。

4.1.1 int 整型数

（1）int 整型

  • 类型 <class 'int'>。

  • Python 整数不分类型，或者说它只有一种类型的整数。Python 整数的取值范围是无限的，不管多大或者多小的数字。

  • 当所用数值超过计算机自身的计算能力时，Python 会自动转用高精度计算（大数计算）。

  • 在 Python 中，可以使用多种进制来表示整数：

    • 十进制形式：常见的整数形式，由 0 ~ 9 共十个数字排列组合而成。

    • 二进制形式：由 0 和 1 两个数字组成，书写时以 0b 或 0B 开头。

    • 八进制形式：由 0 ~ 7 共八个数字组成，书写时以 0o 或 0O 开头。

    • 十六进制形式：由 0 ~ 9 十个数字和 A ~ F（或 a ~ f）六个字母组成，书写时以 0x 或 0X 开头。

  • 为了提高数字的的可读性，Python 3.x 允许使用下划线 _ 作为数字（包括整数和小数）的分隔符。通常每隔三个数字添加一个下划线，类似于英文数字中的逗号。例：num = 1_234.123_456。

（2）bool 布尔型

  • 类型 <class 'bool'>。

  • Python 提供了 bool 类型来表示真（对）或假（错），其中 真（对）使用 True 来代表、假（错）使用 False 来代表。

  • True 和 False 是 Python 中的关键字，要注意字母的大小写。

  • 在 Python 中，bool 是 int 的子类，bool.__base__ 返回的 基类 是 <class 'int'>，True 和 False 可以和数字直接相加，类似于数值 1 和 0。在实际使用中，True == 1、False == 0 会返回 True。

4.1.2 float 浮点型数

  • 类型 <class 'float'>。

  • Python 只有一种小数类型，就是 float。书写时必须包含一个小数点，否则会被当作整数处理。

  • Python 中的小数有两种书写形式：

    • 十进制形式：常见的浮点数形式。

    • 指数形式：写法为 aEn 或 aen，整个表达式等价于 a×10na×10^na×10n。例：2.1E5=2.1×1052.1E5 = 2.1×10^52.1E5=2.1×105。

4.1.3 complex 复数

  • 类型 <class 'complex'>。

  • 复数（complex）是 Python 的内置类型，直接书写即可。

  • 复数由实部（real）和虚部（imag）构成，在 Python 中，复数的虚部以 j 或者 J 作为后缀，具体格式为：a + bj，其中 a 表示实部，b 表示虚部。例：12 + 0.2j。

  • 使用 complex 函数可以表示复数，例：complex(1, 2) 表示复数 1 + 2j。

4.1.4 数字类型转换

  • int(x)：内置函数，将 x 转换为一个整数。

  • float(x)：内置函数，将 x 转换为一个浮点数。

  • complex(x)：内置函数，将 x 转换为一个复数，实数部分为 x，虚数部分为 0。

  • complex(x, y)：内置函数，将 x 和 y 转换到一个复数，实数部分为 x，虚数部分为 y。

4.2 String 字符串

  • Python 中的字符串使用双引号 " " 或单引号 ' ' 创建，双引号和单引号没有区别。

  • 当字符串内容中出现 引号 时，需要对其进行特殊处理，有两种处理方法：

    • 在引号前面添加反斜杠 \ 对引号进行转义，将引号作为普通文本。

    • 使用不同的引号创建字符串，如果字符串内容中出现了单引号，可以使用双引号创建字符串，反之亦然。

  • 要想 换行书写 一个比较长的字符串，可以在行尾添加反斜杠 \ 对字符串内容实现换行。

  • Python 提供了 长字符串 的使用。所谓长字符串，就是可以直接换行（不用加反斜杠 \）书写的字符串。

    • Python 长字符串由三个双引号 """ 或者三个单引号 ''' 创建。

    • 长字符串中的换行、空格、缩进等空白符都会原样输出。

    • 在长字符串中放置单引号或者双引号不会导致解析错误。

    • 如果长字符串没有赋值给任何变量，那么这个长字符串就相当于多行注释。

  • 原始字符串 就是在普通字符串或者长字符串的开头加上 r 前缀。

    • 如果普通字符串的原始字符串中出现引号，程序同样需要对引号进行转义，但是和普通字符串不同的是，此时用于转义的反斜杠会变成字符串内容的一部分。

    • 原始字符串中的反斜杠仍然会对引号进行转义，因此原始字符串的结尾处不能是反斜杠，否则字符串结尾处的引号会被转义，导致字符串不能正确结束。
