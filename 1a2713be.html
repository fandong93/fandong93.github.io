一、顺序结构

  • 顺序结构就是让程序按照从头到尾的顺序依次执行每一条 Python 代码，不重复执行任何代码，也不跳过任何代码。

二、分支结构

  • 分支结构就是构造合适的分支条件和分析程序流程，根据分析程序流程选择适当的分支语句。

  • 在 Python 中，可以使用 if else 语句对条件进行判断，然后根据不同的结果执行不同的代码，这称为选择结构或者分支结构。

  • Python 中的 if else 语句的语法格式：

    • 如果 “condition_1” 为 True，将执行 “statement_block_1” 块语句。

    • 如果 “condition_1” 为 False，将判断 “condition_2”。

    • 如果 “condition_2” 为 True，将执行 “statement_block_2” 块语句。

    • 如果 “condition_2” 为 False，将执行"statement_block_3" 块语句。

    • 注意事项：

      • “表达式（condition）” 可以是一个单一的值或变量，也可以是由运算符组成的复杂语句，形式不限，只要它的结果能转变为一个布尔值即可。

        • 对于数字，Python 会把 0 和 0.0 当做 “False”，把其它值当做 “True”。

        • 对于其它类型，当值为空或 None 时，Python 会当做 “False”，其它情况当做 “True”。

      • if、elif、else 语句的最后要使用冒号 :，表示接下来是满足条件后要执行的代码块。

      • “代码块” 由具有 相同缩进量 的若干条语句组成。

        • Python 是以缩进来标记代码块的，代码块一定要有缩进，没有缩进的不是代码块。

        • 同一个代码块的缩进量要相同，缩进量不同的代码不属于同一个代码块。

      • 一旦某个表达式（if 或 elif）成立，Python 就会执行它后面对应的代码块；如果所有表达式都不成立，那就执行 else 后面的代码块；如果没有 else 部分，那就什么也不执行。

  • Python 中的 if else 语句可以分为三种形式：if 语句、if else 语句和 if elif else 语句。

    if 语句执行流程 if else 语句执行流程 if elif else 语句执行流程

    • if 语句：如果表达式成立（True），执行后面的代码块；如果表达式不成立（False），什么也不执行。

    • if else 语句：如果表达式成立（True），执行 if 后面紧跟的代码块 1；如果表达式不成立（False），执行 else 后面紧跟的代码块 2。如果没有 else 部分，那就什么也不执行。

    • if elif else 语句：Python 从上到下逐个判断表达式（condition）是否成立，如果遇到某个表达式成立（True），就执行后面紧跟的代码块，不再执行其余代码。如果所有的表达式都不成立，就执行 else 后面的代码块。如果没有 else 部分，那就什么也不执行。

      • 不论有多少个 elif 分支，只能执行其中一个分支或一个也不执行，不能同时执行多个分支。

  • if 语句、if else 语句和 if elif else 语句之间可以相互嵌套。

  • 当 if else 条件语句中不进行任何操作时，可以使用空语句 pass 当作占位语句。pass 是 Python 中的关键字，用来让解释器跳过此处，什么都不做。

  • assert 语句，又称 断言语句，可以看做是简配版 if 语句，它用于判断某个表达式的值是否为真，如果为真，则程序可以继续往下执行；反之，Python 解释器会抛出 AssertionError 异常。格式：

  • 在 Python 中没有 switch 语句，但是在 Python 3.10 中新增了 match...case 条件判断，不需要再使用一连串的 if-else。语法格式：

    • match 后的对象会依次与 case 后的表达式进行匹配，如果匹配成功，则执行匹配成功的表达式后面的代码块，否则直接跳过。_ 可以匹配一切，当其他 case 都无法匹配时，执行 case _ 后面的代码块。

3.1 while 循环

  • while 循环的语法格式如下：

    • 首先，判断条件表达式的值，值为真（True）时执行代码块中的语句，否则跳出循环。

    • 当循环体执行完毕后，重新判断条件表达式的值是否为真，若仍为真，则再次执行代码块。

    • 重复上述操作，直到条件表达式的值为假（False），终止循环。

  • 可以通过设置条件表达式永远为 True 来实现无限循环（死循环）。

  • 如果 while 循环体中只有一条语句，可以将该语句与 while 写在同一行。

  • 可以将 while 循环与 else 语句配合使用，当 while 后面的条件表达式为 false 时，执行 else 后面的语句块。语法格式如下：

四、生成器和迭代器

4.1 容器

  • 容器是一种把多个元素组织在一起的数据结构，容器中的元素可以逐个迭代获取，可以用 in，not in 关键字判断元素是否包含在容器中。通常这类数据结构把所有的元素存储在内存中（也有一些特例，并不是所有的元素都放在内存，例：迭代器和生成器对象），我们常用的 string、set、list、tuple、dict 都属于容器对象。

  • 尽管大多数容器都提供了某种方式获取容器中的每一个元素，但这并不是容器本身提供的能力，而是可迭代对象赋予了容器这种能力，当然并不是所有容器都是可迭代的。

    • 可以返回一个迭代器的对象都可以称之为可迭代对象，即实现了 __iter__() 方法的对象都可以称为可迭代对象。通过调用 iter() 方法可以获得一个迭代器（Iterator）。

4.2 迭代器

  • 迭代器是一个带状态的对象，它能在你调用 next() 方法时返回容器中的下一个值，任何实现了 __iter__() 方法和 __next__() 方法的对象都是迭代器。__iter__() 方法返回迭代器自身，__next__() 方法返回容器中的下一个值，如果容器中没有更多元素，则抛出 StopIteration 异常。

  • 迭代器不像其他类型容器一样把所有元素一次性加载到内存，而是以一种延迟计算（lazy evaluation）的方式返回元素。

  • 在使用 for...in... 循环遍历自定义容器对象时，会调用遍历对象的 __iter__() 魔法方法获得一个迭代器，之后再循环调用迭代器对象的 __next__() 方法。__iter__() 只会被调用一次，而 __next__() 会被调用 n 次。

  • 示例：输出斐波那契数列（第一项和第二项为 1，从第三项开始，每一项都等于前两项之和）

4.3 生成器

  • 使用 yield 关键字 返回函数值 的函数叫生成器函数。调用生成器函数时，Python 解释器不会执行函数中的代码，而是会返回一个生成器对象。生成器是一种特殊的迭代器，不过这种迭代器不需要像普通迭代器一样实现 __iter__() 和 __next__() 方法，只需要使用 yield 关键字。

  • 程序每执行到一个 yield 语句就会中断，并返回一个迭代值，下次执行时从 yield 的下一个语句继续执行。

  • 要想使生成器函数得以执行，或者想使执行完 yield 语句立即暂停的程序得以继续执行，有以下 2 种方式：

    • 通过生成器对象调用 next()、send() 或 __next__() 方法。

    • 通过 for 循环遍历生成器。

  • yield 生成器中提供了send 方法，可以给生成器传递参数。每次的 send(params) 都会运行到 yield 语句，但赋值不会执行，只会有返回值，赋值操作在下一次 send(params) 时执行。

  • 生成器表达式：(expression for item in Sequence) 或 (expression for item in Sequence if condition)，使用 () 圆括号将各部分括起来。

    • 可以利用 range 区间、元组、列表、字典和集合等数据类型，快速生成一个满足指定需求的生成器对象。

    • 可是使用 list()、tuple() 或 set() 函数将生成器对象转换成相应的列表、元组或集合。

  • 示例：使用生成器输出斐波那契数列，不需要实现 __iter__() 和 __next__() 方法。

五、推导式

  • 推导式 comprehensions（又称解析式），是 Python 的一种独有特性，是一种可以从一个序列构建另一个新的序列的结构体。共有 3 种推导式：列表推导式、字典推导式 和 集合推导式。

  • 列表推导式格式：[out_exp_res for out_exp in input_list] 或 [out_exp_res for out_exp in input_list if condition]

    • out_exp_res：列表生成元素表达式，可以是有返回值的函数。

    • for out_exp in input_list：迭代 input_list 将 out_exp 传入到 out_exp_res 表达式中。
