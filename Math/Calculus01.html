极限与连续

\quad 极限是微积分中最基本的概念，也是理解导数与积分等概念的基础。

1. 集合

  1. 在表示数集的字母的右上角标上 “*” 来表示该数集内排除 0 的集，标上 “+” 来表示该数集内排除 0 与负数的集，标上 “n” 来表示 n 维数集。例：NNN 表示自然数集合，N+N^+N+ 表示正整数的集合，RRR 表示全体实数的集合，R∗R^*R∗ 表示排除 0 的实数集，R+R^+R+ 表示全体正实数的集合，RnR^nRn 表示 n 维实数集。

  2. 集合 A 的元素数量称为其 基数 或者 势，记为 |A|。在这里复用了绝对值符号。例：集合 A = {1,3,5,7} 的基数为 |A| = 4。

  3. 基数为 有限值 的集合称为 有限集；基数为 无限值 的集合称为 无限集。

  4. 对于两个有限集，如果集合 A 是集合 B 的 真子集，即 A⊂BA\subset BA⊂B，则有 ∣A∣<∣B∣|A|<|B|∣A∣<∣B∣。无限集 的基数为 +∞+\infty+∞，因此不能直接使用这种规则进行基数的比较。

  5. 对于集合 A 和 B ，如果集合 A 中的任意元素 A ，在集合 B 中都有唯一的元素 B 通过某种 映射关系 与之对应，即存在双射关系（Bijection，一对一映射函数）：b=f(a),a∈A,b∈Bb=f(a),a\in A,b\in Bb=f(a),a∈A,b∈B，则称这两个集合的 基数相等（等势）。

  6. 对于集合 A 和 B ，在集合 A 中任意取一个元素 x，在集合 B 中任意取一个元素 y，组成一个有序对 (x,y)，把这样的有序对作为新的元素，它们全体组成的集合称为集合 A 和集合 B 的 笛卡尔乘积（Cartesian product），也称为 直积，记为 A×BA\times BA×B，即 A×B={(x,y)∣x∈A,y∈B}A\times B=\{(x,y)|x\in A,y\in B\}A×B={(x,y)∣x∈A,y∈B}。

  7. 实数集 RRR 与区间 (0,1)(0,1)(0,1) 是等势的。它们之间存在映射：f(x)=11+e−x,x∈R\displaystyle f(x)=\frac{1}{1+e^{-x}},x\in Rf(x)=1+e−x1​,x∈R，此函数被称为 logistic 函数或 sigmoid 函数。

    • 任意长度不为 0 的实数区间都与整个实数集 RRR 等势。

  8. 无限集可进一步分为 可数集（Countable Set）与 不可数集（Uncountable Set）。可数集中的每个元素可以用正整数进行编号，即与正整数集等势。

  9. 可数集的 定义：如果存在从正整数集 N+N^+N+ 到集合 A 的双射关系：f:N+→Af:N^+\rightarrow Af:N+→A，则集合 A 是可数的。

  10. 任意 可数集 在数轴上的 “长度” 为0；不可数集 在数轴上的 “长度” 大于等于0。

    • 可数集的 勒贝格测度 为 0。特别地，有理数集的勒贝格测度为0，尽管有理数集是稠密的。

    • 康托尔集是一个 勒贝格测度 为零的不可数集的例子。
