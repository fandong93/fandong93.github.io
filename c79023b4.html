传统神经网络

发表于2022-10-30|更新于2022-11-21|Deep learning
|字数总计:2k|阅读时长:6分钟|阅读量:

一、深度学习

  • 深度学习是 End-to-End（端到端）的。

  • 深度学习 最大优势 就是简化中间过程，解决了特征工程的问题。

  • 深度学习是真正的去学习什么样的特征是最适合于任务需求的，解决了怎么样去提特征。

  • 深度学习起源于上世纪 60 年代，起初称作 “多层感知机（MLP）”，现在 MLP 是深度学习网络中的一层计算。

  • 由于早期计算机性能不足，所以深度学习没有流行。

  • 1997 年，美国开始将深度学习应用到银行工作中（手写笔记识别），卷积网络出现在了人们视线中。

  • 2009 年，李飞飞号召人们标注 ImageNet 数据集。

  • 2012 年，Alex Krizhevsky 通过 AlexNet 赢得 ImageNet 2012 挑战赛冠军，AlexNet 使用了卷积神经网络，包含八层：前五层是卷积层，之后一些层是最大池化层，最后三层是全连接层。它使用了非饱和的 ReLU 激活函数，显示出比 tanh 和 sigmoid 更好的训练性能。卷积神经网络在 2012 年开始逐渐被人们关注。

  • 2014 年，国内深度学习开始流行。

二、图像分类

  • 计算机视觉：不只是解决传统 RGB 格式数据。

    • 图像数据就是一个矩阵，一张图片可以被表示成三维数组的形式，每个像素的值从 0 到 255。

    • 面临的挑战：照射角度、形状改变、部分遮挡、背景混入，这些问题数据量足够大就可以解决。

      • 真正难题：跨域问题，即一个模型不能满足所有需求。

三、神经网络基础

3.1 线性函数

  • 线性函数：从输入到输出的映射，计算出每个类别的得分。

  • 一共 3072 个像素点，每个像素点对结果影响不同，即每个特征对应了不同的权重参数。

  • 权重值越大表示像素点越重要，权重为正代表促进结果判断，负值代表抑制结果判断。

    • 得出的结果是错的，所以我们需要修改 w 权重参数，x 是固定不可修改的。

3.2 损失函数

  • 损失函数：数据损失，计算 预测结果与正确结果的差异，差异越大，w 越不好，需要大幅调整，反之微调。

    • 研究课题没有创新点，可以创新损失函数。好的论文损失函数设计的都很巧妙。

    • 看论文着重看两点：网络结构设计 和 损失函数设计。

    • 正则化惩罚：防止过拟合，希望权重小一些，惩罚过大的权重。

    • 模型训练参数忽大忽小原因：

      • 数据不干净：离群点、异常点、数据标签不匹配等。

      • batch-size 过小：batch-size 越大，训练越稳定，显存需求越大。

3.3 Softmax 分类器

  • 分类任务中我们想得到的是每个类别的概率值，但是训练模型输出的结果不一定是概率值，所以可以使用 Sigmod 函数将得分值转换成一个概率值，Sigmod 函数在变量值逐渐变大或变小的时候会有 梯度消失 的问题，即梯度接近于 0。

  • 实际在神经网络中，分类问题做法：Softmax 分类

    • 回归任务使用得分值计算损失，分类任务使用概率值计算损失。

3.4 传播

3.4.1 前向传播

  • 前向传播：将上一层的输出作为下一层的输入，并计算下一层的输出，一直到运算到输出层为止。

    • 计算损失的目的：根据损失调整权重参数 W，让损失达到最小，反向传播就是负责做这件事情。

3.4.2 反向传播

  • 反向传播：更新权重参数

    • 不需要推公式，完整写出这部分代码，因为 Pytorch 已经完全封装了反向传播所有的工作。Pytorch 中一行代码。

    • 解决问题：找到一组权重参数，使得损失最小。

      • 第一步：随机初始化权重参数，计算损失。

        • 设置 batch：模型训练时，每次迭代的样本个数，计算 平均损失。batch 越大越好。

          • batch 和 lr 本质上决定了模型的好坏。

      • 第二步：更新权重参数。

        • 找到 更新权重参数的 最优方向，即 “梯度反方向”。沿着梯度反方向更新，所以称作 梯度下降。

        • 设置 沿着梯度反方向更新的 距离，即 “学习率（learning rate，简称 lr）”。

          • warmup 预热策略：当模型刚开始训练时，模型的权重参数是随机初始化的，此时若选择一个较大的学习率，可能带来模型的不稳定（振荡），选择 warmup 预热的方式，可以为刚开始训练的几个 epoch 或者一些 step 设置一个较小的学习率，等模型慢慢趋于稳定后再使用预先设置的学习率进行训练，使得模型收敛速度变得更快，模型效果更佳。

          • 学习率衰减策略：当模型不断训练，权重参数越来越接近最优时，需要将学习率进行衰减，随着迭代次数增加，学习率逐渐减小。

      • 第三步：计算损失，重复执行第二步，直到获得最优权重参数。

        • 如果一次性训练全部样本才更新参数，就将所有样本都丢进去（相当于只有一个 batch），使用 Gradient Descent 梯度下降法进行计算。

        • 如果每计算一个样本，就更新一次参数，使用 Stochastic Gradient Descent 随机梯度下降法进行计算。

        • 如果每次计算一个 batch 才更新参数，则使用 Batch Gradient Descent 批梯度下降法进行计算。

3.4.2 补充

（1）动量

  • 假设某一点 A 经过梯度下降到 B，动量（momentum）可以理解为 A 点梯度下降方向的 惯性，那么动量与 B 点梯度下降方向可以形成一个新的方向，比梯度下降方向更快达到收敛，动量通常设置 0.9 或 0.95，表示保留梯度下降 0.9 倍的力。

3.5 整体架构

  • 每一层做的事情都是在做特征提取，所有神经网络都是为了提取特征。神经网络层数越多代表对特征的转换次数越多。神经网络输出的就是最适合做分类、最适合做预测等的一组特征。

    • 计算机所认识的特征是不可解释的，这也是神经网络不可解释的原因。

    • 全连接层（fully connected layers，FC）就是对输入进行转换得到一组新的特征。

  • 上面讲述的神经网络中一层全连接层的特征转换是线性的，输出结果也是线性的，再连接一层全连接后还是线性的。但是我们希望神经网络是非线性的，所以我们在神经网络中加入了非线性函数：激活函数。

    • Relu 函数将小于 0 的特征置 0，大于 0 的特征再进行学习。

    • 数据做预处理的目的是让模型更好的学习数据，经过一层学习之后得到一组新的特征，不能保证这一组新的特征与源特征分布一致，所以对新的特征再进行一次预处理，以此类推，对后续全连接层每次得到的新特征都进行预处理，做预处理的层被称为 Batch Normalization 层，常加在激活函数之前。

3.6 数据预处理

  • 标准化：

    • 第一步：去均值，即均值变为 0，数据以原点为中心对称。

    • 第二步：各维度取值范围尽可能相同。

3.7 参数初始化

  • 通常我们使用随机策略来进行参数初始化。

3.8 DROP-OUT

  • 随机杀死一些神经元，即随机去除一些特征。用于防止过拟合。
