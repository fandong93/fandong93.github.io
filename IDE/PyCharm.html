一、PyCharm 的安装与激活

1.1 PyCharm 的安装

（1）Windows

  • 官网：https://www.jetbrains.com/pycharm/download/other.html 下载 Version 2021.3.3 的 2021.3.3 - Windows x64 (exe) 版本。

  • 双击安装：

    • 建议勾选 Create Destop Shortcut 和 Create Associations。

1.2 PyCharm 的激活

  • 运行 PyCharm，出现 Import PyCharm Settings 页面，不导入之前的 PyCharm 配置，勾选 Do not import settings，点击 OK。

  • 启动页面

  • PyCharm 运行成功后，会弹出下面的对话框，提示我们需要先登录 JetBrains 账户才能使用，这里我们先不管，先点击 Exit 退出，准备开始运行激活脚本。

  • 开始激活

    • 自行下载 ja-netfilter-all 激活文件夹，更名为 JetBrains 后放置到电脑上某个位置。

    • 修改 PyCharm 安装目录下，文件夹 bin 中的 pycharm64.exe.vmoptions 配置文件。

    • 文件末尾加入激活补丁的位置：-javaagent:D:\JetBrains\ja-netfilter.jar=jetbrains

    • 启动 PyCharm，选择 Activation code，填入激活码。

    • 点击 Activate 进行激活后，可以看到激活到 2099 年。

二、PyCharm 的详细配置

2.1 进入设置界面

2.1.1 方式一

  • 欢迎界面下，点击 Customize →\rightarrow→ All settings... 进入设置界面。

2.1.2 方式二

  • 项目界面下，点击菜单 File →\rightarrow→ Settings... 进入设置界面。

2.2 关闭 PyCharm 自动更新

  • 进入设置界面 →\rightarrow→ Appearance & Behavior →\rightarrow→ System Settings →\rightarrow→ Updates，取消 Check IDE updates for: 的勾选 →\rightarrow→ OK。

2.3 配置项目默认路径

  • 进入设置界面 →\rightarrow→ Appearance & Behavior →\rightarrow→ System Settings →\rightarrow→ Default project directory 选择项目默认路径。

2.4 创建项目

2.4.1 方式一

  • 欢迎界面下，点击 Projects →\rightarrow→ New Project。

  • 创建 Pure Python 项目，Location 位置填写项目路径，最后的文件夹即为项目名：pythonProject。选择 Previously configured interpreter →\rightarrow→ 配置 Python 编译器所在路径 →\rightarrow→ Create

  • 选择 System Interpreter，配置解释器所在路径 →\rightarrow→ OK

  • 创建成功界面如下：

2.4.2 方式二

  • 项目界面 下，点击菜单 File →\rightarrow→ New Project...，然后按照上述步骤进行创建。

2.5 配置 Python 解释器

  • 进入设置界面 →\rightarrow→ Project: pythonProject →\rightarrow→ Python Interpreter →\rightarrow→ Add...

2.6 配置编码

  • 进入设置界面 →\rightarrow→ Editor →\rightarrow→ File Encodings →\rightarrow→ Global Encodings: 和 Project Encodings: 选择 UTF-8 →\rightarrow→ Default encoding for properties files: 选择 UTF-8 →\rightarrow→ 勾选 Transparent native-to-ascll conversion →\rightarrow→ Create UTF-8 files: 选择 with NO BOM →\rightarrow→ Apply

2.7 主题

2.7.1 代码主题

  • 安装主题

    • 下载地址：http://www.themesmap.com/theme.html?t=down

    • 下载主题：Ladies Night 2

  • 导入主题：打开设置界面 →\rightarrow→ Editor →\rightarrow→ Color Scheme →\rightarrow→ 齿轮设置 →\rightarrow→ Import Scheme... →\rightarrow→ 选择 Ladies Night 2 主题所在路径 →\rightarrow→ OK

  • 设置主题：Color Scheme →\rightarrow→ Python →\rightarrow→ 选择 Ladies Night 2 主题 →\rightarrow→ OK

2.7.2 IDE 主题

（1）方式一

  • 欢迎界面点击 Customize →\rightarrow→ Color theme，选择要设置的主题。

（2）方式二

  • 进入设置界面 →\rightarrow→ Appearance & Behavior →\rightarrow→ Appearance →\rightarrow→ Theme，选择要设置的主题。

    • 获取更多主题：Get more themes →\rightarrow→ 选择要安装的主题 Install →\rightarrow→ Accept

2.8 设置字体

2.8.1 IDEA 界面字体

（1）方式一

  • 在欢迎界面下，点击 Customize →\rightarrow→ 在 Accessibility 中进行设置 IDEA 界面字体大小。

（2）方式二

  • 进入设置界面 →\rightarrow→ Appearance & Behavior →\rightarrow→ Appearance →\rightarrow→ 设置 IDEA 界面字体大小。

2.8.2 代码编辑界面字体大小

  • 进入设置界面 →\rightarrow→ Editor →\rightarrow→ Font →\rightarrow→ 设置代码编辑界面字体大小及行距 →\rightarrow→ OK

2.8.3 Ctrl + 鼠标滚轮修改字体大小

  • 进入设置界面 →\rightarrow→ Editor →\rightarrow→ General →\rightarrow→ 勾选 Change font size with Ctrl+Mouse Wheel in: →\rightarrow→ 自选 Active editor 或 All editors

2.8.4 控制台字体

  • 默认情况下无需调整，会跟随代码编辑界面使用一致的设置，如需自定义则可以进入设置界面 →\rightarrow→ Editor →\rightarrow→ Color Scheme →\rightarrow→ Console Font →\rightarrow→ 勾选 Use console font instead of the default，然后自定义控制台字体

2.9 行号和方法分隔符

  • 进入设置界面 →\rightarrow→ Editor →\rightarrow→ General →\rightarrow→ Appearance →\rightarrow→ 勾选 Show line numbers 显示行号，勾选 Show method separators 显示方法分隔符

2.10 忽略大小写匹配

  • 进入设置界面 →\rightarrow→ Editor →\rightarrow→ General →\rightarrow→ Code Completion →\rightarrow→ 取消勾选 Match case →\rightarrow→ OK

2.11 配置自动导包

  • 进入设置界面 →\rightarrow→ Editor →\rightarrow→ General →\rightarrow→ Auto Import →\rightarrow→ 勾选 Show auto-import tooltip 和 from <module> import <name>

2.12 安装第三方包

  • 进入设置界面 →\rightarrow→ Project: pythonProject →\rightarrow→ Python Interpreter →\rightarrow→ +

  • 搜索要安装的包 →\rightarrow→ Install Package

Ctrl + W 增加选中的代码范围

Alt + Enter 代码快速修正

Ctrl + Alt + O 最佳化导入
Ctrl + Alt + I 自动缩进
Tab 缩进
Shift + Tab 删除缩进
Ctrl + X / Shift + Delete 剪切当前行或选定的代码块到剪贴板
Ctrl + C / Ctrl + Insert 复制当前行或选定的代码块到剪贴板
Ctrl + V / Shift + Insert 从剪贴板粘贴
Ctrl + Shift + V 打开历史粘贴板
Ctrl + D 复制当前行到下一行 或 复制选定代码到后面

Ctrl + Shift + J 多行代码合并为一行
Shift + Enter 快速向下另起一行

快捷键 功能
CTRL + F 文件中查找
Ctrl + Shift + F
（与 win10 简繁转换快捷键冲突）		 全局查找
Ctrl + R 文件中替换
Ctrl + Shift + R 全局替换
Ctrl + F3 查找变量下一次使用的位置
Shift + F3 查找前一个

Alt + F7 查询变量所有用过的地方
Ctrl + F7 查询变量在文件中用过的地方
Ctrl + Shift + F7 高亮显式文件中变量用到的地方
Ctrl + Alt + F7 显示变量所有用到的地方

3.3 代码运行（Running）

快捷键 功能
Alt + Shift + F9 选择程序文件调试
Alt + Shift + F10 选择程序文件运行
Shift + F9 调试当前文件代码
Shift + F10 运行当前文件代码
Ctrl + Shift + F10 运行当前编辑区的程序文件

快捷键 功能
F7 单步执行，进入子函数
F8 单步执行，不进入子函数
F9 继续恢复程序，直接运行到下一断点处
Alt + Shift + F7 单步执行时，遇到子函数就进入并且继续单步执行，不会进入到源码中
Shift + F8 跳出当前函数体，返回到调用此函数的地方
Alt + F9 运行到光标所在位置处
Ctrl + Alt + F9 强制运行到光标所在位置，即在断点处不停顿
Alt + F8 验证表达式
Ctrl + Alt + F8 快速验证表达式
Ctrl + F8 添加 / 取消断点
Ctrl + Shift + F8 编辑 / 显示所有断点

Ctrl + [ / Ctrl + ] 跳转到代码块开始 / 结束
Ctrl + F12 弹出文件结构

Ctrl + Shift + H 方法层次结构
Ctrl + Alt + H 调用层次结构
F2 / Shift + F2 快速跳转到下一条 / 上一条报错的代码
F4 / Ctrl + Enter 编辑资源、查看资源
Alt + Home 显示导航条F11书签开关
