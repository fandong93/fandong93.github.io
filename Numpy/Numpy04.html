\quad Numpy 是 Python 中科学计算的核心库，NumPy 这个词来源于 Numerical 和 Python 两个单词。它提供了一个高性能的多维数组对象，以及大量的库函数和操作，可以帮助程序员轻松地进行数值计算，广泛应用于机器学习模型、图像处理和计算机图形学、数学任务等领域。

一、NumPy 数学函数

\quad NumPy 中包含了大量的数学函数，它们用于执行各种数学运算，其中包括加减乘除、三角函数、舍入函数等等。下面对它们做详细讲解。

1.1 加(add)、减(subtract)、乘(multiply)、除(divide)

1.2 sqrt 函数

  • numpy.sqrt(arr) 函数返回 arr 数组每个元素的正平方根。

1.3 三角函数

1.2.1 sin(x)、cos(x)、tan(x)

    • sin() 和 cos() 函数值域 [-1，1]，tan() 函数值域为 R\pmb{R}RRR 实数。

1.2.2 arcsin(x)、arccos(x)、arctan(x)

  • numpy.arcsin(x)，numpy.arccos(x) 和 numpy.arctan(x) 函数返回给定角度的反三角函数。

    • numpy.arcsin(x) 函数和 numpy.arccos(x) 函数接收 [-1，1] 范围的参数，numpy.arcsin(x) 函数返回 [−π2,π2]\pmb{\displaystyle[-\frac{π}{2},\frac{π}{2}]}[−2π​,2π​]​[−2π​,2π​]​​[−2π​,2π​] 范围的弧度值，numpy.arccos(x) 函数返回 [0,π]\pmb{\displaystyle[0,π]}[0,π]​[0,π]​​[0,π] 范围的弧度值。

1.3 舍入函数

1.3.1 numpy.around()

  • numpy.around(arr, decimals) 函数返回一个十进制数，并将数值四舍五入到指定的小数位上。

    • arr：代表要输入的数组；

    • decimals：要舍入到的小数位，默认 0。如果为负数 -n，将小数点左侧 n 位四舍五入（当 n 大于小数点左侧位数时返回 0.）。

  • 实例：

  • numpy.ceil(x) 函数表示对数组中的每个元素 向上取整数，即返回 不小于 数组中每个元素值的最大整数。示例：

1.4 其他算术运算

1.4.1 numpy.reciprocal()

  • numpy.reciprocal(a) 函数对数组 a 中的每个元素 取倒数，并以数组的形式将它们返回。此函数不适用于整数，由于 Python 处理 整除（//） 的方式，对于绝对值大于 1 的整数参数，返回值为 0；对于整数零参数，返回值将抛出异常：RuntimeWarning: invalid value encountered in reciprocal。示例：

1.4.2 numpy.power()

  • numpy.power(a, b) 函数将数组 a 中的元素作为 底数，将数组 b 中的元素作为 幂，以数组形式返回两者的计算结果。在做 power 运算时，输入数组 a 和 b 必须具有相同的形状，或者符合数组的广播规则，才可以执行运算。示例：

1.4.3 numpy.mod()

  • numpy.mod(x1, x2) 函数返回 x1，x2 两个数组相对应位置上元素相除后的 余数，它与 numpy.remainder(x1, x2) 的作用相同 。

    • 如果 a 和 d 是两个自然数，d 非零，可以证明存在两个唯一的整数 q 和 r，满足 a = qd + r 且 0 <= |r| < |d|。其中，q 被称为 商，r 被称为 余数。

      • 在计算机语言中，同号 的整数运算，所有语言都遵循尽量让商小的原则，所以 12 mod 5 和 -12 mod -5 是一样的方式，结果差一个符号，分别是 2 和 -2。

      • 但是在 异号 的整数运算中，C 和 Java 都是尽可能让商 q 更大（例：-12 mod 5 的结果对应的是商 q = -2，余 r = -2），而 Python 则是会让商尽可能的小（例：-12 mod 5 的结果对应的是商 q = -3，余 r = 3）。

    • numpy.mod 等同于 Python 中的取模运算符 x1 % x2，符号和除数 x2 保持一致。

  • 示例：

2.2 numpy.argmin()

  • numpy.argmin(arr, axis) 函数用于从数组中查找 最小元素的索引，默认为沿数组的展平版本计算 最小值的索引。如果设置了参数 axis，则沿指定的轴 axis 查找数组中元素的 最小值的索引，以数组形式返回。axis=0 对应着最外层的 []，axis=1 对应第 2 外层的 []，以此类推，axis=n 对应第 n 外层的 []。示例：

2.3 numpy.amax()

  • numpy.amax(arr, axis) 函数用于从数组中查找 最大元素，默认为计算所有元素的 最大值。如果设置了参数 axis，则沿指定的轴 axis 查找数组中元素的 最大值，以数组形式返回。axis=0 对应着最外层的 []，axis=1 对应第2外层的[]，以此类推，axis=n 对应第 n 外层的 []。示例：

2.4 numpy.argmax()

  • numpy.argmax(arr, axis) 函数用于从数组中查找 最大元素的索引，默认为沿数组的展平版本计算 最大值的索引。如果设置了参数 axis，则沿指定的轴 axis 查找数组中元素的 最大值的索引，以数组形式返回。axis=0 对应着最外层的 []，axis=1 对应第 2 外层的 []，以此类推，axis=n 对应第 n 外层的 []。示例：

2.5 numpy.ptp()

  • numpy.ptp(arr, axis) 函数用于计算数组中元素 最值之差，也就是（最大值 - 最小值），默认为计算所有元素的 最值之差。如果设置了参数 axis，则沿指定的轴 axis 查找数组中元素的 最值之差，以数组形式返回。axis=0 对应着最外层的 []，axis=1 对应第 2 外层的 []，以此类推，axis=n 对应第 n 外层的 []。示例：

2.6 numpy.percentile()

2.10 numpy.var()

  • numpy.var(a, axis) 函数用于沿指定轴 axis 计算方差。默认方差计算展平的数组，否则计算指定的轴。

  • 统计中的方差（样本方差）是每个样本与全体样本的平均数差值的平方和的平均数，即 mean((x - x.mean())** 2)，方差描述了随机变量对于数学期望（平均值）的偏离程度。

四、NumPy 排序函数

\quad NumPy 提供了多种排序函数，这些排序函数可以实现不同的排序算法。排序算法特征主要体现在以下四个方面：执行速度，最坏情况下的复杂度，所需的工作空间以及算法的稳定性。下表列举了三种排序算法：

\quad 假设在待排序的序列中，存在多个具有相同的关键字的记录，若经过排序，这些记录的相对次序保持不变，即在原序列中，r[i] = r[j]，且 r[i] 在 r[j] 之前，而在排序后的序列中，r[i] 仍在 r[j] 之前，则称这种排序算法是 稳定的，否则称为不稳定的。

4.1 numpy.sort()

  • sort(arr, axis=-1, kind='quicksort', order=None)[source] 函数对数组进行排序，并返回一个数组副本。

    • arr：要排序的数组。

    • axis：沿着指定轴进行排序，如果没有指定 axis，默认在 最后一个轴 上排序。

    • kind：可选项 {'quicksort'，'mergesort'，'heapsort'}，默认为 quicksort（快速排序）。

    • order：表示要排序的字段，可以是 str 或 str 的列表。
