\quad Numpy 是 Python 中科学计算的核心库，NumPy 这个词来源于 Numerical 和 Python 两个单词。它提供了一个高性能的多维数组对象，以及大量的库函数和操作，可以帮助程序员轻松地进行数值计算，广泛应用于机器学习模型、图像处理和计算机图形学、数学任务等领域。

一、NumPy 位运算

1.1 bitwise_and 函数

  • bitwise_and 函数对两个数组中 整数 的二进制形式执行 位与运算，返回一个 新数组。格式：numpy.bitwise_and(x1, x2)。

  • 示例：

1.3 invert 函数

  • invert 函数对数组中整数做按位取反运算，也就是 0 变成 1，1 变为 0，返回一个 新数组。格式：numpy.invert(x)。注：计算机中都是使用补码存储数字，所以按位取反都是取补码的相反值。

    • 对于有符号二进制数，其最高位为 0，表示正数；最高位为 1，表示负数。

    • 补码：正数的补码和原码，反码相同；负数的补码则是在原码的基础上符号位保持不变，其他位取反，然后在末尾加 1。

    • 二进制负数转十进制：对二进制负数按位取反再加 1，然后加上负号，结果就是二进制负数转化成的十进制数。

    • invert 函数等价于 bitwise_not 函数，调用 bitwise_not 函数实际是调用 invert 函数。

  • 实例：将由 [-5,5] 范围的随机数组成的 3 行 4 列矩阵按位取反。

    • 如图所示：-4 的补码 11111100，按位取反 00000011，转十进制即为 3；5的补码即原码 00000101，按位取反 11111010，转十进制即为 -6。

1.4 left_shift 函数

  • left_shift(arr, step) 函数将数组 arr 中的每个元素的二进制形式 向左移动 step 位，右侧以 0 填充，返回一个 新数组。（左移二进制数 nnn 位相当于原数字乘以 2n2^n2n）

    • step 可以是整数或整数数组，当 step 是数组时，要求其与 arr 数组能够实现广播。

  • 实例：

1.5 right_shift 函数

  • right_shift(arr, step) 函数将数组 arr 中的每个元素的二进制形式 向右移动 step 位，左侧以 0 填充，返回一个 新数组。（右移二进制数 nnn 位相当于原数字除以 2n2^n2n，向下取整）。

    • step 可以是整数或整数数组，当 step 是数组时，要求其与 arr 数组能够实现广播。

    • 注：负数位数不够会变成 -1，正数位数不够会变成 0。

  • 实例：将由 [-5,5] 范围的随机数组成的 3 行 4 列矩阵每个元素右移动 2 位。

2.15 numpy.char.partition()

  • numpy.char.partition(arr, sep) 函数通过指定分割符 sep 对字符串数组 arr 进行分割。如果元素包含多个分割符，则从 最左边第一次 出现的分割符开始分割，仅分割一次，返回一个 新数组，数组中的每个元素包含三个元素：sep 前的字符串、分割字符 sep 和 sep 后的字符；如果找不到分隔符，则返回三个元素： 字符串本身 及 两个空字符串。

    • sep：可以为一个分隔符或分隔符数组，如果是分隔符数组，则要求其与字符串数组 arr 可以实现广播。

    • sep：可以为一个分隔符或分隔符数组，如果是分隔符数组，则要求其与字符串数组 arr 可以实现广播。

2.21 numpy.char.replace()

  • numpy.char.replace(arr, old, new, count=None) 函数可以使用新字符串 new 来替换原字符串数组 arr 中的子字符串 old。

    • count：int 类型，如果指定该值，则只会替换每个元素中出现的前 count 个子字符串 old。

2.22 numpy.char.translate()

  • numpy.char.translate(arr, table, deletechars=None) 函数将字符串数组 arr 按照 字符 映射转换表 table 进行转换。如果指定了 deletechars 列表，则先移除 deletechars 包含的字符然后进行转换。注：转换的是字符，而不是由字符组成的字符串。

    • 字符映射转换表 table 通过 str.maketrans(intab,outtab[,delchars]) 方法转换而来，该方法返回一个由转换前后字符的 ASCII 值组成的键值对所构造的字典，如果设置了 delchars 参数，则 delchars 中每个转换后的值为 None。maketrans 方法的参数如下：

      • intab：需要转换的字符组成的字符串。

      • outtab：转换的目标字符组成的字符串。

      • delchars：可选参数，表示要删除的字符组成的字符串。

  • 实例：将字符 a 和 b 转换为 A 和 B，删除字符 c 和 d。

2.23 numpy.char.encode()

  • numpy.char.encode(arr, encoding=None, errors=None) 函数用于编码操作，字符串数组 arr 中的每个元素依次调用 str.encode，可以使用 Python 标准库中的编解码器。

    • arr：要处理的 str 或 unicode 数组。

    • encoding：编码名称，可选项，str 类型，默认编码为 utf-8。

    • errors：指定如何处理编码错误，可选项，str 类型。默认为 'strict'，意为编码错误引起一个 UnicodeError。 其他可能得值有 'ignore'、'replace'、'xmlcharrefreplace'、'backslashreplace' 以及通过 codecs.register_error() 注册的任何值。

    • arr：要处理的 str 或 unicode 数组。

    • errors：指定如何处理编码错误，可选项，str 类型。默认为 'strict'，意为编码错误引起一个 UnicodeError。 其他可能得值有 'ignore'、'replace'、'xmlcharrefreplace'、'backslashreplace' 以及通过 codecs.register_error() 注册的任何值。
