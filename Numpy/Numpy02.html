\quad Numpy 是 Python 中科学计算的核心库，NumPy 这个词来源于 Numerical 和 Python 两个单词。它提供了一个高性能的多维数组对象，以及大量的库函数和操作，可以帮助程序员轻松地进行数值计算，广泛应用于机器学习模型、图像处理和计算机图形学、数学任务等领域。

\quad Numpy 中包含了一些函数用于处理数组，大概可分为以下几类：

  • 数组遍历

  • 改变数组形状

  • 数组转置

  • 修改数组维度

  • 连接数组

  • 分割数组

  • 数组元素增删改查

一、数组遍历

1.1 普通迭代

  • ndarray 的普通迭代与 Python 及其他语言的迭代方式一样，N 维数组就要用 N 层的 for 循环。

  • 实例：遍历二维数组。

1.2 nditer 多维迭代器

1.2.1 基本使用

  • NumPy 提供了一个 nditer 迭代器对象，它可以配合 for 循环完成对数组元素的遍历。

  • 实例：使用 arange() 函数创建一个 3 行 4 列数组，并使用 nditer 生成迭代器对象。

1.2.2 order 参数：指定访问元素的顺序

  • 在内存中，Numpy 数组提供了两种存储数据的方式，分别是 C-order（行优先顺序）与 Fortrant-order（列优先顺序），默认是行优先顺序。

  • 可以通过 nditer 对象的 order 参数来指定访问元素的顺序。默认以内存中元素的顺序（order='K'），即创建 ndarray 时指定的顺序。

    • for x in np.nditer(a, order='F')：Fortran order，即是列序优先；

    • for x in np.nditer(a, order='C')：C order，即是行序优先。

  • 实例：分别以行序优先和列序优先输出数组 a。

  • 需要注意的一个重要问题是，顺序的选择是为了匹配数组的内存布局，这样做是为了提高访问效率，我们可以通过迭代一个数组的转置来了解这一点。

  • a 和 a.T 的遍历顺序是一样的，也就是他们在内存中的存储顺序也是一样的，但是 a.T.copy() 的遍历结果是不同的，那是因为它和前两种的存储方式是不一样的，默认是按行访问。

1.2.3 op_flags 参数：迭代时修改元素的值

  • nditer 对象提供了一个可选参数 op_flags，它表示能否在遍历数组时对元素进行修改。它提供了三种模式，如下所示：

    • readonly（默认）：只读模式，遍历时不能修改数组中的元素。
    • readwrite：读写模式，遍历时可以修改元素值。
    • writeonly：只写模式，遍历时可以修改元素值。

  • 实例：将 nditer 对象的 op_flags 参数设置为 readwrite，修改 a 数组元素值。

1.2.4 flags 参数

1.外部循环

flags=[‘external_loop’]

  • 当指定 flags=[‘external_loop’] 时，将返回一维数组而并非单个元素。具体来说，当 ndarray 的存储顺序和遍历顺序一致时，将所有元素组成一个一维数组返回；当 ndarray 的存储顺序和遍历顺序不一致时，返回每次遍历的一维数组。

  • 实例 1：ndarray 的存储顺序和遍历顺序一致时，将所有元素组成一个一维数组返回。

  • 实例 2：ndarray 的存储顺序和遍历顺序不一致时，返回每次遍历的一维数组。

二、改变数组形状

2.1 reshape 函数

  • reshape 函数返回一个修改 ndarray 形状后的视图（view）对象。格式：ndarray.reshape(self, newshape, order = 'C') 或 numpy.reshape(ndarray, newshape, order='C')。

    • ndarray：要操作的数组。

    • newshape：取值为整数或整数元组，新的形状应当兼容原有形状。如果是整数，则结果将是该长度的一维数组。 某一个维度的尺寸可以为 -1，在这种情况下，该值是根据数组的元素个数和其余维度长度来推断的。

    • order：使用 C / F / A 索引顺序读取 ndarray 的元素，并使用此索引顺序将元素放入重塑的数组中，默认 C。C 表示按行索引顺序读取/写入元素，F 表示按列索引顺序读取/写入元素，C 和 F 选项不考虑底层数组的内存布局，仅参考索引顺序。A 表示如果 ndarray 在内存中的存储结构是 F order 连续的，则按 F order 读取/写入数据，否则都按 C order 读取/写入数据。

  • 实例：使用 arange 生成 ndarray 对象，然后使用 reshape 函数修改形状。

2.2 flatten 函数

  • flatten 函数将 ndarray 数组展开成一维的数组，该数组是原始数组的 拷贝，对拷贝所做的修改不会影响原始数组。格式：ndarray.flatten(self, order='C')

    • order：指定数组展开的方向，默认 “C”。C 表示行方向；F 表示列方向；A 表示如果 ndarray 数组在内存中的存储结构是 F order 连续的，则按列方向展开，否则都按行展开；K 表示按照存储结构原顺序展开。

  • 实例：将 3 行 4 列数组展开成一维数组。

2.3 ravel 函数

  • ravel 函数将多维数组 ndarray 中的元素以一维数组的形式展开，该方法返回原始数组的 视图（view）或 拷贝（copy）。格式：ndarray.ravel(self, order='C') 或 numpy.ravel(ndarray, order='C')

    • ndarray：要操作的数组。

    • order：指定数组展开的方向，默认 “C”。C 表示行方向；F 表示列方向；A 表示如果 ndarray 数组在内存中的存储结构是 F order 连续的，则按列方向展开，否则都按行展开；K 表示按照存储结构原顺序展开。

    • 注：如果 order 与 ndarray 数组的存储结构顺序不同，返回的是原始数组的拷贝。

  • 实例：将 3 行 4 列数组展开成一维数组。

    • axes：可选，取值为元组或列表。默认情况下，反转维度，否则根据给定数组对轴进行排列。

3.2 ndarray.T 属性

  • ndarray.T 与 numpy.transpose() 无参方法相同，即反转维度。

3.3 rollaxis 函数

  • rollaxis 函数将数组指定的轴 axis 向后滚动，直到位于给定的位置，返回数组的 视图。格式：numpy.rollaxis(ndarray, axis, start)。

    • ndarray：要操作的数组。

    • axis：要向后滚动的轴，取值为整数，其它轴的相对位置不会改变。axis = 0 对应着最外层的 []，axis = 1 对应第 2 外层的 []，以此类推，axis=n 对应第 n 外层的 []。

    • start：表示指定的轴要滚动到的位置，默认 0。

      • 如果 axis 参数值大于或等于 start，则 axis 轴从后向前滚动，直到 start 位置；

      • 如果 axis 参数值小于 start，则 axis 轴从前向后滚动，直到 start 的前一个位置，即 start-1 位置。

3.4 swapaxes 函数

  • swapaxes 函数用于交换数组的两个轴，返回数组的 视图。格式：ndarray.swapaxes(self, axis1, axis2) 或 numpy.swapaxes(ndarray, axis1, axis2)。

    • ndarray：要操作的数组。

    • axis1：第一个轴的整数。axis1 = 0 对应最外层的 []，axis1 = 1 对应第 2 外层的 []，以此类推，axis1 = n 对应第 n 外层的 []。

    • axis2：第二个轴的整数。axis2 = 0 对应最外层的 []，axis2 = 1 对应第 2 外层的 []，以此类推，axis2 = n 对应第 n 外层的 []。

四、修改数组维度

3.1 broadcast 函数

  • broadcast 函数模仿广播机制，返回一个 numpy.broadcast 对象，该对象封装了参数之间相互广播的结果，numpy.broadcast 对象中每一个元素都是一个元组，元组中包含每个参数广播后对应位置的值。numpy.broadcast 对象具有 shape、nd (ndim)、size 和 iters 属性，可作为迭代器使用。格式：numpy.broadcast(ndarray1, ndarray2, ...)。

    • ndarray1, ndarray2, ...：相互广播的 ndarray 数组。

  • 实例 1：numpy.broadcast 对象用作迭代器。

3.2 broadcast_to 函数

  • broadcast_to 函数将数组广播到新形状，返回一个 只读视图。如果新形状不符合 numpy 的广播规则，则抛出 ValueError 异常。格式：numpy.broadcast_to(array, shape, subok=False)

    • array：要广播的数组。

    • shape：要广播到的新形状，取值为整数元组。

    • subok：如果为 True，则新创建的数组将使用 array 的类型，否则它将是基类 ndarray。默认为 False。

3.3 expand_dims 函数

  • expand_dims 函数通过在指定位置插入新的轴来扩展数组形状，返回数组的 视图。格式：numpy.expand_dims(arr, axis)

    • arr：要扩展的数组。

    • axis：新轴插入的位置，取值为整数或整数元组。axis=0 对应着最外层的 []，axis = 1 对应第 2 外层的 []，以此类推，axis = n 对应第 n 外层的 []。

3.4 squeeze 函数

  • squeeze 函数用于删除 ndarray 数组中维度为 1 的项，返回数组的 视图。格式：ndarray.squeeze(self, axis) 或 numpy.squeeze(arr, axis)。

    • arr：要删除维度的数组。

    • axis：取值为整数或整数元组，用于指定需要删除的维度。若指定维度，则值必须为 1，否则会抛出 ValueError 异常，若为 None，则删除数组中所有值为 1 的维度。axis = 0对应着最外层的 []，axis = 1 对应第 2 外层的 []，以此类推，axis = n 对应第 n 外层的 []。

四、连接与分割数组

4.1 连接数组

4.1.1 concatenate 函数

  • concatenate 函数在一个 已经存在的轴 上连接相同形状的两个或多个数组。格式：numpy.concatenate((a1, a2, ...), axis)。

    • a1, a2, ...：表示一系列相同形状的数组。注：指定轴 axis 的值可不同。

    • axis：沿着该参数指定的轴连接数组，默认为 0。axis = 0 对应着最外层的 []，axis = 1 对应第 2 外层的 []，以此类推，axis = n 对应第 n 外层的 []。

4.1.2 stack 函数

  • stack 函数用于沿 新轴 连接数组序列。格式：numpy.stack((a1, a2, ...), axis)。注：stack 会增加一个维度，而 concatenate 不会增加维度。

    • (a1, a2, ...)：相同形状的数组序列。

    • axis：沿着该参数指定的轴连接数组，默认为 0。axis = 0 对应着最外层的 []，axis = 1 对应第 2 外层的 []，以此类推，axis = n 对应第 n 外层的 []。

4.1.3 hstack 函数

  • hstack 函数是 stack 函数的变体，它通过水平堆叠（按列堆叠）来生成数组，相当于沿着第二个轴进行连接。格式：numpy.hstack(tup)。注：hstack 函数不会增加维度，因此 hstack 函数类似于 np.concatenate((a,b), axis = 1)。

    • tup：相同形状需要堆叠的数组组成的元组。注：第二个轴的值可不同。

  • 实例：

4.1.4 vstack 函数

  • vstack 函数是 stack 函数的变体，它通过垂直堆叠（按行堆叠）来生成数组，相当于沿着第一个轴进行连接。格式：numpy.vstack(tup)。注：vstack 函数不会增加维度，因此 vstack 函数类似于：np.concatenate((a,b),axis = 0)。

    • tup：相同形状需要堆叠的数组组成的元组。注：第一个轴的值可不同。

4.2 分割数组

4.2.1 split 函数

  • split 函数沿特定的轴将数组分割为子数组，返回 子数组列表，格式：numpy.split(ary, indices_or_sections, axis=0)

    • ary：要被分割的数组。

    • indices_or_sections：如果是一个整数，就用该数平均切分，不能均分则会抛出异常：ValueError: array split does not result in an equal division；如果是一维数组，则为沿轴切分的位置，如果索引沿轴超出数组的维数，则将相应返回一个空的子数组。例：参数是数组 [3,7]，被切割的数组会被分为 [3],[3,7],[7:] 三个部分。

    • axis：设置沿其分割的轴，默认为 0。axis = 0 对应着最外层的 [ ]，axis = 1 对应第 2 外层的 [ ]，以此类推，axis = n 对应第 n 外层的 [ ]。

5.1 resize 函数

  • numpy.resize 函数返回指定形状的新数组。如果新数组大小大于原始大小，则使用原始数组中的元素进行填充。格式：numpy.resize(arr, new_shape)

    • arr：要改变形状的数组。

    • new_shape：取值为整数或整数元组，用于指定数组的新形状。

  • numpy.resize 函数可以放大或者缩小原数组的形状：放大时，使用 原数组元素 补全剩余元素；缩小时，直接丢弃多余元素。而 reshape 函数要求 reshape 前后元素个数相同，否则会抛出异常。

    • new_shape：取值为整数或整数元组，用于指定数组的新形状。

    • refcheck：取值为布尔值，可选，表示是否检查引用计数，默认为 True。

5.2 append 函数

  • append 函数将值 values 附加到 arr 数组的 副本 尾部，如果未指定 axis，则返回的是一个一维数组。格式：numpy.append(arr, values, axis=None)

    • arr：要被追加元素的数组。

    • values：如果指定 axis，则 values 必须是与 arr 数组形状相同的数组。注：指定轴 axis 的值可不同；如果未指定 axis，则 values 可以是任何形状，并在使用前展平。

    • axis：沿着哪个轴进行追加，如果为 None，则 arr 将被展平。axis = 0 对应着最外层的 [ ]，axis = 1 对应第 2 外层的 [ ]，以此类推，axis = n 对应第 n 外层的 [ ]。

  • 实例：原数组 arr 的形状为 (2,3,4)，指定 axis = 1，则轴 0、2 需要与原数组保持一致。

5.3 insert 函数

  • insert 函数表示沿指定的轴，在给定索引值的前一个位置插入值，并返回一个 新数组。如果没有提供轴，则输入数组被展开为一维数组。格式：numpy.insert(arr, obj, values, axis)。

    • arr：要被插入元素的数组。

    • obj：表示索引值，在该索引值之前插入 values 值，可以是整数、切片或整数序列。

      • 如果指定 axis，不能大于轴 axis 的长度。

      • 如果未指定 axis，不能大于所有元素的数量。

    • values：要插入的值。如果传入的是数组，那在除了 axis 之外的轴要符合广播规则。

      • 注：当 values 传入的是数组时，在广播之前会首先将指定的 axis 放在轴 0 处，然后再进行广播。例：values 形状为 (1,2,3)，指定 axis = 1，则会先将形状变为 (2,1,3)。然后判断 2 和 3 是否符合广播规则。

    • axis：沿着它插入的轴，如果为 None，则 arr 将被展平。axis=0对应着最外层的[]，axis=1对应第2外层的[]，以此类推，axis=n对应第n外层的[]。如果未提供axis，则输入数组会被展开且values只能是单行序列。

5.4 delete 函数

  • delete 函数表示从输入数组中删除指定的子数组，并返回一个 新数组。它与 insert 函数相似，若不提供 axis 参数，则输入数组被展开为一维数组。格式：numpy.delete(arr, obj, axis=None)。

    • arr：要删除元素的数组。

    • obj：表示要沿指定轴删除的子数组的索引，可以是整数、切片或整数序列。

      • 如果指定 axis，不能大于轴 axis 的长度。

      • 如果未指定 axis，不能大于所有元素的数量。

    • axis：沿着哪条轴删除子数组，如果为 None，则 arr 将被展平。axis = 0 对应最外层的[]，axis=1对应第2外层的[]，以此类推，axis=n对应第n外层的[]。

5.5 argwhere 函数

  • argwhere 函数默认返回数组中非 0 元素的索引，若是多维数组则返回由行索引和列索引组成的索引坐标数组。格式：numpy.argwhere(arr)。

    • arr：可以是一个数组，返回数组中非 0 元素的索引；也可以是要索引数组的条件。

5.6 unique 函数

  • unique 函数用于去除数组中的重复元素，返回去重并排序的 新数组；。格式：numpy.unique(arr, return_index=False, return_inverse=False, return_counts=False, axis=None)。

    • arr：要去重的数组。除非指定了轴，否则如果输入数组不是一维数组会返回一个无重复元素并升序的一维列表。

    • return_index：是否返回由每个唯一值在原数组第一次出现的索引组成的数组，默认 False。

      • 注：返回的索引是原数组展平后元素所在的索引。

    • return_inverse：是否返回由原数组元素在新数组中的索引组成的数组，默认 False。

    • return_counts：是否返回由新数组中的每个唯一值在原数组的出现次数组成的数组，默认 False。

    • axis：要去重的轴，如果为 None，则 arr 将被展平。axis=0对应着最外层的[]，axis=1对应第2外层的[]，以此类推，axis=n对应第n外层的[]。
