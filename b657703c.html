一、函数定义

  • 函数是一段封装好的，可重复使用的，用来实现单一或相关联功能的代码。

  • 函数定义的语法：

    • 关键字：

      • def：定义函数的关键字，所有函数都以此开头，后接函数名。

      • return：结束函数，返回一个值给调用者，不填写返回值的 return 相当于返回 None。

      • pass：如果函数不做任何事情，或者函数体中的某些分支语句不做任何事情，可以使用 pass 语句用作占位语句，保持程序结构的完整性。

    • 函数名：一个符合 Python 语法的标识符，建议做到见名知意。

    • 形参列表：形参列表中的参数用逗号 , 分开，声明了调用函数时需要传入的参数。如果函数不接收任何参数，参数列表就为空 ()。

      • 注：函数不需要参数，也要保留括号 “()”，否则 Python 解释器报错 “invaild syntax”。

    • 函数体：定义一个函数功能的所有代码组成的整体，以冒号 : 开头并缩进。第一行语句可以选择性地使用注释书写函数说明。

      • 注：可以使用 pass 定义空函数。

    • 返回值：函数被调用后得出的结果，可以是表达式，也可以是变量。Python 函数支持多返回值。

  • 示例：

  • 通过调用 Python 的 help() 内置函数或 __doc__ 属性，我们可以查看某个函数的使用说明文档。

    • 示例：help(rectangleArea) 或 rectangleArea.__doc__

二、函数调用

2.1 函数调用

  • 函数调用也就是执行函数体，可以在一个函数中调用其他函数。

  • 函数调用的基本语法格式：[返回值] = 函数名([实参列表])

    • 返回值：如果调用的函数存在返回值，我们可以通过一个变量来接收，也可以不接收。

    • 函数名：要调用的函数的名称。

    • 实参列表：创建函数时要求传入的各个形参的值。实参数量与形参数量要保持一致。

2.2 参数传递

2.2.1 形参 VS 实参

  • 形参（形式参数）：在定义函数时，函数名后面括号中的参数可以看做是一个占位符，它没有数据，只能等到函数被调用时接收传递进来的数据，所以称为形式参数，简称形参。

  • 实参（实际参数）：在调用函数时，函数名后面括号中的参数包含了真实存在的数据，会被函数内部的函数体使用，所以称为实际参数，简称实参。

2.2.2 可变类型 VS 不可变类型

  • 可变类型（可变数据类型）：当该数据类型对应的变量的 值发生变化 时，如果变量的 内存地址不发生改变，那么这个变量对应的数据类型就是 可变数据类型。

    • list（列表）、dict（字典）

  • 不可变类型（不可变数据类型）：当该数据类型对应的变量的 值发生变化 时，如果变量的 内存地址发生改变，那么这个变量对应的数据类型就是 不可变数据类型。

    • numbers （数值类型）、string（字符串）、tuple（元组）。

2.2.3 传递方式

  • 值传递：在调用函数时将实参复制一份传递到函数中，这样在函数中如果对参数进行修改，将不会影响到实参。

  • 引用传递：在调用函数时将实参的地址传递到函数中，那么在函数中对参数所进行的修改，将影响到实参。

  • 在 python 中，一切皆对象，数字、字符串、元组、列表、字典、函数、方法、类、模块等都是对象，所以 python 参数传递采用的是 “传对象引用” 的方式，这种方式相当于值传递和引用传递的一种综合。

    • 如果函数收到的是一个可变对象的引用，就能修改对象的原始值，相当于 “引用传递”。

    • 如果函数收到的是一个不可变对象的引用，就不能直接修改原始对象，相当于 “值传递”。

2.2.4 参数类别

（1）位置参数（必需参数）

  • 调用函数时，按照传递的实际参数的 位置 识别对应的形式参数。

    • 在调用函数时，传递的实际参数的数量必须和形式参数的 数量 一致，否则会抛出 TypeError 异常。

    • 在调用函数时，传递的实际参数的位置必须和形式参数的 位置 一致，否则会抛出 TypeError 异常（数据类型不一致）或得到错误的结果。

（2）关键字参数

  • 调用函数时，按照传递的实际参数的 名字 识别对应的形式参数。

    • 通过此方式指定函数实参时，不再需要与形参的位置完全一致，只需参数名对应即可。

    • 在调用函数时，可以同时使用位置参数和关键字参数，但是位置参数必须放在关键字参数之前。
