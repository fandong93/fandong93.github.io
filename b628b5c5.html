一、TeX 介绍

  • TeX 是由著名的计算机科学家 Donald Ervin Knuth（高德纳）于 1978 年开始研发的 排版系统，核心由 Pascal 语言编写，包含大量排版命令。

  • 高德纳最早开始自行编写 TeX 的原因，是因为当时的电脑排版技术十分粗糙，已经影响到他的巨著《计算机程序设计艺术》的印刷品质。他以典型的黑客思维模式，决定自行编写一个排版软件：TeX。

  • TeX 是一个系统，就不能单纯说它是 “一种编程语言” 或 “一个程序”，实际上它由 排版语言、编译器（引擎）以及一些相关 可执行程序 构成。用户编写 .tex 源文件，TeX 编译器将其编译为二进制机器码，并存储在 .dvi 文件中。

  • TeX 真正生成的其实是 .dvi 文件（而不是 PDF），DVI（Device Independent）译为设备无关。这种文件精确描述了所有页面元素的样式以及各自坐标，并且绝对固定并无法被更改，所以任何读取它的输出设备（显示器、打印机等）显示出来的效果一定是相同的。

  • TeX 拥有几百条原始指令，可以将这些原始的 TeX 指令重新组合为一个新的指令，这种功能称为宏。

  • plain TeX 是一组宏命令集合，也就是说在 TeX 原始指令上，扩展了许多新的指令。

  • TeX 没有软件运行界面，是命令行格式的。

二、LaTeX 介绍

  • LaTeX 是一种基于 ΤΕΧ 的排版系统，由美国计算机学家 莱斯利·兰伯特（Leslie Lamport）在 20 世纪 80 年代初期开发，利用这种排版的处理，即使使用者没有排版和程序设计的知识也可以充分发挥由 TEX 所提供的强大功能，不必一一亲自去设计或校对，能在几天，甚至几小时内生成很多具有书籍品质的印刷品。对于生成复杂表格和数学公式，这一点表现得尤为突出。因此它非常适用于生成高印刷质量的科技和数学、物理文档。这个系统同样适用于生成从简单的信件到完整书籍的所有其他种类的文档。

  • LaTeX 系统是一种可以处理排版和渲染的标记语言。

  • LaTeX 是基于 TeX 的一组宏集，相当于对 TeX 进行了一次封装，屏蔽一些细节，提供了很多预设的模版和样式，能够让我们能够更加容易的编写文档。

三、TeX Live

3.1 概述

  • TeX Live 是由国际 TeX 用户组（TeX Users Group，TUG）整理和发布的 TeX 软件发行套装，主要 用于 LaTex 编译，包含与 TeX 系统相关的各种程序、编辑与查看工具、常用宏包及文档、常用字体及多国语言支持。TeX Live 是许多 Linux/Unix 系统（例：Fedora、Debian、Ubuntu、Gentoo 等）默认或推荐的 TeX 套装，同时也支持包括 Windows 和 Mac OS X 等在内的其它操作系统。TeX Live 是开发状态最为活跃的 TeX 发行版 之一，保持着每年一版的更新频率。TeX Live 属于免费软件。

  • 由于 TEX/LATEX 并不是单独的程序，现在的 TEX 系统都是复杂的软件包，里面包含各种排版的引擎、编译脚本、格式转换工具、管理界面、配置文件、支持工具、字体及数以千计的宏包和文档。TEX 发行版就是把所有这样的部件都集合起来，打包发布的软件。

3.2 下载离线安装包

  • 官网：https://tug.org/texlive/ 点击 On DVD →\rightarrow→ 点击 downloading the TeX Live ISO image and burning your own DVD →\rightarrow→ 点击 download from a nearby CTAN mirror; →\rightarrow→ 点击 texlive2022.iso

  • 清华大学镜像文件：https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/Images/

3.3 安装 TeX Live

  • 解压 texlive2022.iso 压缩包，然后右键 “以管理员身份运行” 打开其中的 install-tl-windows.bat 文件。

  • 点击 Advanced

  • 修改安装目录 →\rightarrow→ 点击安装

3.4 验证安装

  • 打开命令提示符，输入 tex -version，如果出现版本号即说明安装成功。

四、TeXstudio

4.1 概述

  • TeXstudio 是一款开源的跨平台 LaTeX 编辑软件，作为 LaTeX 的集成开发环境，TeXstudio 为用户提供了交互式拼写检查、代码折叠、语法高亮等特性但并 没有集成编译 LaTeX 文件的功能，用户需自行安装 LaTeX 编译软件。

  • TeXstudio 能在 Windows、Unix、BSD、Linux 以及 Mac OS 上运行。

4.2 下载安装包

  • 官网：https://www.texstudio.org/

  • 开源软件仓库：https://texstudio.sourceforge.net/

4.3 配置

  • 设置中文界面：Options →\rightarrow→ Configure TeXstudio... →\rightarrow→ General →\rightarrow→ Language →\rightarrow→ zh_CN →\rightarrow→ OK

  • 指定 Commands 命令的执行程序：设置为 TeXlive 安装路径下对应的 exe 文件。

  • 设置编码：将默认字体编码改为 UTF-8。Editor →\rightarrow→ Default Font Encoding →\rightarrow→ UTF-8

  • 添加行号：Editor →\rightarrow→ Show Line Numbers →\rightarrow→ All Line Numbers

  • 设置编译器：中文论文，使用 xelatex 编译器，英文论文，使用 pdflatex 编译器，文献工具，使用 BibTeX。

五、JabRef

5.1 概述

  • JabRef 是一套开放源代码、处理 BibTeX 格式的文献管理软件，提供了简易操作的界面来编辑 BibTeX 档案。功能包含从网络上的科学数据库汇入资料，以及整理和搜寻 BibTeX 档案等。

  • BibTeX 是一种用于管理文献、产生文献目录的文件格式，通常与 LaTeX 一起使用，其后缀为 .bib。

5.2 下载安装包

  • 官网：https://www.jabref.org/
