1. 安装 gnome-tweak-tool

  • 安装 gnome-tweak-tool：

  • 打开 Tweaks →\rightarrow→ Appearance →\rightarrow→ Shell 前面会有感叹号。

2. 去掉 Shell 感叹号

  • 安装 gnome-shell-extensions：

  • 启用 User themes 扩展：打开 Tweaks →\rightarrow→ Extensions →\rightarrow→ User themes 按钮点开。（如果没有 User themes 扩展选项，需要重启 Ubuntu 系统）

  • 重启 Tweaks 后，Shell 前面的感叹号消失了。

3. 配置主题 GTK Themes

  • 进入 https://www.gnome-look.org/browse/ 网站的 GTK3/4 Themes 分类根据个人喜好下载 GTK 主题。

  • 作者下载了 McMojave-Dark 主题。

  • 解压文件到 /usr/share/themes 文件夹：

  • 打开 Tweaks →\rightarrow→ Appearance →\rightarrow→ 在 Shell 和 Applications 中选择主题即可。

4. 配置图标 Icon Themes

  • 进入 https://www.gnome-look.org/browse/ 网站的Full Icon Themes分类根据个人喜好下载图标主题。

  • 作者下载了 McMojave-circle 主题。

  • 解压文件到/usr/share/icons文件夹：

  • 打开 Tweaks →\rightarrow→ Appearance →\rightarrow→ 在 Icons 中选择主题即可。

5. 配置光标 Cursor

  • 进入 https://www.gnome-look.org/browse/ 网站的 Cursors 分类根据个人喜好下载光标主题。

  • 作者下载了 McMojave cursors 主题。

  • 解压文件到 /usr/share/icons 文件夹：

  • 打开 Tweaks →\rightarrow→ Appearance →\rightarrow→ 在 Cursor 中选择主题即可。

6. 配置壁纸 Background

  • 4k4k4k、8k8k8k 壁纸资源：

    • 极简壁纸

    • WALLPAPER ABYSS

    • wallhaven

  • 下载图片到 /usr/share/backgrounds 文件夹。

  • 打开 Tweaks →\rightarrow→ Appearance →\rightarrow→ 在 Background 中选择照片 image 即可。

7. 配置登录背景 login

  • 下载 ubuntu-20.04-change-gdm-background 文件，并使用 -P 指定下载路径：

  • 给 change-gdm-background 文件赋予执行的权限：

  • 修改背景图片：

    • 其中 /usr/share/backgrounds/lamber.jpg 为背景图片所在路径。

  • 屏蔽紫色过渡动画：对于登录系统的紫色过渡动画，就是输入密码回车，到进入系统这个过程的紫色动画，可以通过 gnome 的扩展 extension 进行屏蔽，名字是 Good Bye GDM3 Login Screen to Desktop Flick for Ubuntu 20.04 only 。

8. 文件夹改色助手 Color Folder

  • Color Folder 可以帮助快速改变文件夹的颜色使它与其他文件夹看起来不同。
