1. 换源

  • 备份原来的源：sudo cp /etc/apt/sources.list /etc/apt/sources_init.list

  • 打开 sources.list 文件：sudo gedit /etc/apt/sources.list

  • 更换 sources.list 文件内容：阿里源 https://developer.aliyun.com/mirror/

  • 更新源：

    • 如果报错：使用 sudo apt-get -f install 修复损坏的软件包，尝试卸载出错的软件包，重新安装正确版本的软件。

  • 其他源：

    • 清华源：https://mirrors.tuna.tsinghua.edu.cn/

    • 网易163源：http://mirrors.163.com/

    • 中科大源：http://mirrors.ustc.edu.cn/

2. 安装谷歌输入法

  • shell 输入：sudo apt-get install fcitx-googlepinyin

  • Settings →\rightarrow→ Region & Language →\rightarrow→ Manage Installed Languages

    • 第一次打开会显示 The language support is not installed completely，点击 Install 开始安装。安装成功之后就能进入 Language Support 界面，然后 Keyboard input method system 选择 fcitx，操作完成之后进行 重启。

  • 输入法配置：重启之后在右上角状态栏点击 键盘图标，在下拉菜单点击 Configure 进入配置界面。点击 Input Method Configuration 左下角的+号，进入到 Add input method 界面。取消 Only Show Current Language 选项的勾选，输入 goo 即可搜索到 Google Pinyin 输入法，然后选择 Google Pinyin 并点击 OK 确认即可。切换输入法的快捷键是 ctrl + space。

3. 卸载 libreoffice

  • 卸载

  • apt-get 安装的软件包会存储在 /var/cache/apt/archives/ 和 /var/cache/apt/archives/partial/ 两个目录下，长期使用会占用硬盘空间。sudo apt-get clean 指令就是删除掉这两个目录中的软件包，除了已经被锁定的文件。sudo apt-get autoclean 同样是删除掉这两个目录下的软件包，不同的是autoclean 只删除不能被再次下载的软件包，所以说 sudo apt-get clean 删除的更彻底。

4. 安装 wps

  • 下载安装包：https://linux.wps.cn/

  • 使用 dpkg 来安装 wps：sudo dpkg -i wps-office_11.1.0.10702_amd64.deb

  • 设置桌面快捷方式：进入 /usr/share/applications 文件夹，将下列后缀为 .desktop 的文件复制到桌面，然后在 .desktop 的文件上右键，选择 Allow Launching 即可。

    1. 下载字体：https://pan.baidu.com/s/1nkmjLmFytGUBYc4dWXa7Dw 密码：6aml

    2. 解压到指定文件夹：tar -zxvf 要解压的文件名 -C 要解压到的路径

    3. 生成索引信息：

    4. 更新字体：sudo fc-cache -fv

5. 安装 Google Chrome

  • 使用 wget 来下载 Google Chrome 的 .deb 软件包：

    • 下载到 指定目录：wget -P 指定目录 文件下载地址

    • 下载到 当前目录 并 重命名：wget -O 指定文件名 文件下载地址

    • 下载到 指定目录 并 重命名：wget -O 指定目录/指定文件名 文件下载地址

  • 使用 dpkg 来安装 Google Chrome：sudo dpkg -i google-chrome.deb

12. 安装 git

  • git 具有三个配置文件：

    • 版本库级 配置文件（.git/config），查看版本库级的 config：git config --local -l；

    • 全局级 配置文件（~/.gitconfig ），查看全局级的 config：git config --global -l；

    • 系统级 配置文件（/etc/gitconfig ），查看系统级的 config：git config --system -l；

  • 查看当前生效的配置，git config -l。

14. 卸载软件

14.1 显示已安装软件

  • 打开终端，输入 sudo dpkg --list 命令，显示出已安装的所有软件。

14.2 软件卸载

  • 在终端上输入命令 sudo apt-get --purge remove package-name。

    • --purge 为可选项这个属性是将软件及其配置文件全部删除。

    • sudo apt-get remove package-name 仅删除软件，不删除配置文件。
