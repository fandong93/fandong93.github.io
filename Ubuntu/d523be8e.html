1. win10 分配空闲磁盘

\quad 使用 DiskGeniusDiskGeniusDiskGenius 数据恢复软件对磁盘空间进行拆分。也可以在 Win10 左下角开始图标上鼠标右键，在右键菜单中找到磁盘管理，打开即可。作者电脑为250G 固态硬盘（实际 238G）及 1T 机械硬盘（实际 932G），所以我将固态硬盘划分 38.2G 空闲，机械硬盘划分 150G 空闲。

2. 检查 windows10 的引导方式

  • Win + R 运行，输入 msinfo32，回车弹出系统信息，查看 BIOS 模式，本文适用于 UEFI 引导。

3. 制作 U 盘启动盘

  • 使用 RufusRufusRufus 工具创建可引导 USB 闪存盘。分区类型选择 GPT，引导类型选择要安装的 Ubuntu 系统。

4. 安装 ubuntu 20.04 系统

4.1 进入 bios

  • 重启 Win10，按 F2 / F10 / F12（不同机型，进入 bios 的按键不同）进入 bios，选择 UEFI 进入 Ubuntu 20.04 系统安装。

4.2 禁用驱动

  • 在 Ubuntu 选项按键盘 e 键进入编辑。

  • 在 quiet splash 后加入 nomodeset，告诉内核在系统启动之前不启动视频驱动程序，而改用 BIOS 模式。然后按键盘 F10 进入 Ubuntu20.04 安装界面。

4.3.5 Installation type

  • 想要手动对 Ubuntu 系统进行分区，要选择 Something else 选项，如果不要手动分区，选择 Install Ubuntu alongside windows Boot Manager 选项即可自动安装 Ubuntu 系统。

  • 注：千万不要选择 Erase disk and install Ubuntu，不然 Win10 系统会被删除。

4.3.6 磁盘分区

  • 作者将机械硬盘划分 16G\pmb{16G}16G16G16G 给 /tmp 分区（如果想使用 Ubuntu 系统的休眠功能，需要划分 /tmp 分区），剩余空间都划分给 /home 分区。作者电脑内存为 16G\pmb{16G}16G16G16G，所以没有划分 swap 交换分区，若读者需要可以划分 swap 交换分区，空间大小与内存大小相同即可。

  • 作者将固态硬盘划分为 efi 分区：1G\pmb{1G}1G1G1G、/boot 分区：1G\pmb{1G}1G1G1G 和 / 分区：剩余空间。Device for boot loader installation 要选择 efi分区 所在位置。

  • Install Now →\rightarrow→ Continue

  • 时区选择 Shanghai, China。

  • 设置系统用户名、密码。

  • 等待安装。

  • 最后会弹出对话框，选择 Restart Now 即可。

5. 重启 ubuntu 20.04 系统

5.1 禁用视频驱动

  • 在 Ubuntu 选项按键盘 e 键进入编辑。

  • 在 quiet splash 后加入 nomodeset，告诉内核在系统启动之前不启动视频驱动程序，而改用 BIOS 模式，然后按键盘 F10 进入 Ubuntu20.04 系统。

5.2 永久禁用 nouveau 驱动

5.2.1 方法一

  • 打开 blacklist.conf 文件：sudo gedit /etc/modprobe.d/blacklist.conf

  • 文本最后加入：

  • 更新 initramfs：sudo update-initramfs -u。

  • 重启系统：reboot。

  • 验证：lsmod | grep nouveau，如果没有输出，说明禁用 nouveau 成功。

5.2.1 方法二

  • 打开 grub 文件：sudo gedit /etc/default/grub。

  • 修改 grub 文件：GRUB_CMDLINE_LINUX_DEFAULT="quiet splash" 换成 GRUB_CMDLINE_LINUX_DEFAULT="quiet splash nomodeset"。

  • 更新 grub 文件：sudo update-grub。

  • 重启：reboot。

5.3 换源

  • 备份原来的源：sudo cp /etc/apt/sources.list /etc/apt/sources_init.list

  • 打开 sources.list 文件：sudo gedit /etc/apt/sources.list
